Description

你知道吗？不同的生物是不共用停车场的，比如人类有人类停车场，喵星人有喵星人停车场，皮卡丘有皮卡丘停车场……但是有这么一种生物他在现实世界里是喵星人，在虚拟世界里是皮卡丘，所以他既可以在喵星人停车场停车，也可以在皮卡丘停车场停车，同时他还有未卜先知的能力。

不同生物的停车场之所以不共用是因为每种生物修建停车场和在停车场停车的习惯不同。喵星人停车场只有一个出入口，进入停车场的车辆必须等在它后面进入的车辆都离开了停车场才可以驶出——没错，喵星人停车场是一个栈。皮卡丘停车场有一个入口和一个出口，进入停车场的车辆必须等它前面进入停车场的车辆都离开了才可以驶出——皮卡丘停车场是一个队列，你又猜对了吧？

今天一只像上文中提到既是喵星人又是皮卡丘的生物来到了WH市。WH市有喵星人停车场和皮卡丘停车场各一个。停车前这只生物运用他的超能力得知了这两个停车场当天其他所有车辆的停车时间和想要离开的时刻。当然，他也知道自己要停车和要离开的时间。请问，这只奇妙的生物应该在哪个停车场里停车才能让他离开时的等待时间比较短呢？

Input

输入的第一行有一个整数 t，表示有 t 组测试数据。
对于每组测试数据：
第一行有四个数据：n, m, pt, lt，分别表示喵星人停车场的车辆数、皮卡丘停车场的车辆数、该生物的停车时刻和它想要要离开的时刻。
第二行有 n 个数据，分别表示喵星人停车场第 i 辆车的停车时刻 mpt[i]。
第三行有 n 个数据，分别表示喵星人停车场第 i 辆车想要离开的时刻 mlt[i]。
第四行有 m 个数据，分别表示皮卡丘停车场第 i 辆车的停车时刻 ppt[i]。
第五行有 m 个数据，分别表示皮卡丘停车场第 i 辆车想要离开的时刻 plt[i]。
数据范围：0 ＜ n, m ＜ 86399，0 ≤ pt, lt, mpt[i], mlt[i], ppt[i], plt[i] ≤ 86399
注意：所有时刻均以“一天中的秒数”（0~86399）表示。你可以假设所有的车辆都在当天内停车和离开，任何一个时刻停车场总是先让车辆离开再让车辆进入且一个时刻最多有一辆车进入，车辆进入和离开停车场都是瞬间完成的，在两个停车场的等待时间不会相等。

Output

对于每组测试数据输出一行，如果该生物要将车停在喵星人停车场则输出“miaowu~”；如果停在皮卡丘停车场输出“pika~”。

Sample Input

1
2 2 25200 43200
21600 28800
39600 54000
21600 28800
39600 54000

Sample Output

pika~
