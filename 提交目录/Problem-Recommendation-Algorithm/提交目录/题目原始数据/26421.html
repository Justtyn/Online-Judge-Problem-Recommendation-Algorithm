Description

经过十一年的抗战，蓦然终于来到神往已久的大学，大家都知道大学是一个容易让人堕落的地方。对于学生的上课迟到是不能容忍的，这所学校的老师对上课迟到有如下惩罚：
数学老师： 把自己的头发的数量，全部数清楚，汇报出奇数还是偶数，数三次数字都不一致的，直接剃度送去少林寺出家。
英文老师： 去给自己取1000个英文名，禁止重复，性别不可混淆，否则将重复的名字抄写1000遍，并赐予最新名字为：毕癞头be later!
物理老师： 让他一只脚站在讲台上，金鸡独立40分钟，好让他知道什么叫做压强，脚给与桌子的压力，会让他记得作用力等与反作用力，脚大一点或许压强小一点了。
蓦然来到学校后也开始慢慢放松了，不是翘课就是迟到，正是这样他被各科老师惩罚了不少。。。哎，苦逼啊！
又是一觉睡到自然醒，蓦然看看手机，7:40。坏了，闹钟咋没响，真是该响时不响，不该响时它却乱响。
“啊，今天语文课，老师上次说了今天要点名”，室友猛然从床上坐起，惊叫道。
麻利地、、、
当然，语文老师没有那么严厉了，漂亮，温柔的老师对蓦然和他的室友说：你们如果在一分钟内把下面的题目解决，我就大人不记小人过，饶了你们。
题中有两个字符串，要求找到第二个字符串在第一个字符串中第一次出现的位置（至少出现一次）。
蓦然木然，以前学的语文知识都还给老师了。。。聪明的你能帮帮他吗？

Input

第一行一个整数t（1=<t<=30），有t组数据，接下来每组数据有两行。

每行一个字符串，只包含小写字母，无空格，长度小于100。

Output


对于每组数据，输出第二个字符串在第一个字符串中第一次出现的位置，即匹配完成后第二个字符串的第一个字符在第一个字符串中的位置。

acm

8
