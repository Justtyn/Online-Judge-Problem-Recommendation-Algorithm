Problem 4416 --H、DS®可乐

4416: H、DS®可乐

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 0  Solved: 0

Description

 DS们最近发现市面上出了一种新牌子的可乐，名字和他们一样，于是他们非常支持，纷纷买了很多集装箱的可乐回家喝。DS牌可乐富含各种物质，有益于身体的各种矿物质、维生维、DHA、ANA，也有氯水、水银、苏丹红、塑化剂、三聚氰胺等天朝食品常见添加剂，喝了这些可乐，将会导致DS身体发生一些变化（也许他很可能变成绿巨人）。每听可乐中含有各种物质的含量不同，有的喝了体质会变好，有的喝了体质会变弱。

每个DS都有一样的癖好，把自己买回来的可乐一听接一听摆成一个头尾相接的圆环，并且从此以后不再重新排列这些可乐，然后选择其中一听开始喝，从此以后按顺时钟顺序把所有可乐喝完。
DS们开始喝这种可乐以前的体质可能各不相同，每听可乐对体质的影响不同，DS界有一个统一的单位——MZ，可以将体质量化。当体质小于或等于0 MZ时，DS将会死亡。
DS可能还没把买回来的所有可乐喝完就挂了，也有可能喝完所有可乐他还健康地活着。这也许和最开始选择哪一听喝有关，也许无论开始时选择哪一听都可能是死或者活。

Input

 输入有若干行，每行最开始有两个整数，某个DS喝可乐之前的体力值整数H(0 < H <= 20000)和该DS买的可乐的总听数N(0 < N <= 5000)，然后有N个数，表示该DS买回可乐并且摆好之后，随便以其中一听开始，顺时针顺序列出的各听可乐对体质影响的整数值hi(-5000<=hi<=5000)，hi为正数即表示列出的第i听可乐对体质有增强作用，负数则为削弱，也可能对体质没有影响。

当输入的H和N同时为0时，程序结束，不必对这行数据进行求解。

Output

 对应每一个输入，输出一行结果，仅有一个正整数，当DS把买回来的可乐摆好之后，有几种可能把所有买到的可乐喝完之后还活着（也就是体质值大于0MZ）。

Sample Input

20 5 -10 -9 6 -5 2
1 6 -6 6 -6 6 -6 6
0 0

Sample Output

4
3
