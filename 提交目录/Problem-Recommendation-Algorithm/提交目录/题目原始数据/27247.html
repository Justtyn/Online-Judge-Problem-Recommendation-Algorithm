Description

　小E在成功地破解箱子的密码后,发现箱子里面放着一个游戏机,游戏机上显示四个大字“回文游戏”。所谓回文嘛，就是正着读，反着读都一样。那么回文游戏要怎么玩呢？游戏机允许每次输入一个整数i(1≤i≤N-1，N为字符串长度)，表示可以交换字符i和字符(i+1)，小E一想，这真是太简单了。这时小C进来，并没有充满愤怒，而是笑着告诉小E“你还没调难度呢！”，于是小C把难度调world class级别，字符串的长度可以有10^6，小E马上就瘫了，虽然容易但这样玩要玩多久呀……请你帮助小E完成这个简单的任务。

Input

输入文件包含一个大写字母字符串表示被操作的串

Output

输出文件包含一个整数ans,表示最少经过ans步操作，可以将原串变成回文串。如果不行则输出-1

Sample Input

SHLLZSHZS

Sample Output

4
【样例解释】
首先交换L和Z变成  SHLZLSHZS
再交换L和Z变成    SHZLLSHZS
交换L和S变成      SHZLSLHZS
最后交换H和Z变成  SHZLSLZHS

【数据范围】
1.	对于10%的数据，N≤1000
2.	对于20%的数据，N≤5000
3.	对于40%的数据，N≤50000
4.	对于100%的数据，N≤1000000
