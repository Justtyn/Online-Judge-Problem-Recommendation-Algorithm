Description

选课啦，选课啦~！BNUEP一学期一度的选课时间又到了，小佳是信息技术学院出了名的好学生，每次到了选课时间就特别纠结，她想选尽量多的学分。当然，这个专业课是学院定好，小佳在这方面无能为力，于是她只能在选修课上花更多的功夫了。

         经过一年大学的熏陶，小佳已经知道了选修课都是浮云~学不到什么实质性的东西，于是她只是抱着混学分的思想去选课的，所以她有自己的选课规则。

 

1. 她想选尽量多的学分来完成选修部分，让自己大三有更多时间钻研自己的专业课。

2. 有必修课的时间不能选选修课，因为那样是违背小佳好好学习的精神的。

 

由于小佳的不淡定，在教务系统上乱选，终于弄得自己的课表一片凌乱~Orz~~~

小佳现在很想知道自己究竟是最多能修多少个选修学分。

聪明的你能写个程序告诉小佳她最多能选多少个选修学分么？

Input

首先输入一个整数P，代表有P组测试数据。

对于每组测试数据，首先用一行给出两整数 N M，代表将有N门必修和M门可以选择的选修课。保证每门课每周只有一节。

接下来的N行，每行给出两个整数d t（1<=d<=5, 1<=t<=7），代表这个必修课在周 D 的第 T 节上课。必修课之间没有课程冲突。

接下来的M行，每行给出三个整数 S D T（S <= 5 , 1<=D<=5, 1<=T<=7）,分别代表每一门可供选择的选修课的学分S，在周 D 的第 T 节上课。

Output

每组数据单独输出一行，小佳能选的最多的选修学分是多少（小佳保佑做出这题的人不挂选修~）。

Sample Input

1<br/>1 1<br/>1 2<br/>3 2 4<br/>

Sample Output

3<br/>

HINT

显而易见的，你不能放弃你的必修课来选一门选修课。
