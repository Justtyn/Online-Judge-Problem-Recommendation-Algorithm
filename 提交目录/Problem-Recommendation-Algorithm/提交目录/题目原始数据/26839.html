Description

区域内有一些羊和一些狼。每个动物都有自己的私人领域，他们总是呆在自己的地区。

然而，狼吃羊的杀戮和自然，当狼看到自己相邻区域有羊时就会去吃羊（但是隔着的区域狼就看不到羊），所以

现在区域经理决定建造一些栅栏分开的两个物种。

现场可以通过地图N×M表示，每个单位的地图可能被动物占领。如果是自住的，这意味着这是个人的私人领域。

篱笆只能建立在单元区域的边界。

现在的问题是，在栅栏的最小长度是多少？

Input

每个测试案例的第一行包含两个整数N和M（2≤n，m＜＝100）。

以下N行每一个包含M个整数。每个整数可以是1，2或0。如果是1，这个区域是由狼占领了；如果是2，该地区是

以一只羊占用；如果是0，这个区域不占用。

Output

对于每个测试案例，你的程序应该只打印一行包含一个整数，代表篱笆的最小长度。

Sample Input

3 2
1 2
2 2
1 1
2 3
1 0 2
1 2 2

Sample Output

4
1
