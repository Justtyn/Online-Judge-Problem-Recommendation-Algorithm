Description

“球球又去拱北地下商场了......”

球球是个很喜欢逛商场的女生，尤其是拱北的地下商场。但球球是个天生缺乏方向感的家伙，在商场里她会经常走着走着就找不着北了。

最近，球球终于有了一台iphone，她希望有一个程序，可以让她在逛商场的时候，随时地知道从当前自己所处的位置走出商场至少还要多少步。

也许，你就能帮她实现这样一个需求了。现在，给出地下商场的(矩形)平面地图，以及球球当前所处的位置，请计算告诉她从当前所处位置开始，她至少需要多少步才能走出商场。

Input

第一行输入两个整数W和H，分别表示地下商场平面图的宽和高。

接下来输入H行，每行输入W个字符。字符‘.’表示商场里面可行走的道路，字符‘#’表示墙壁(不可经过和穿越)， 字符‘*’表示球球当前所在的位置点。

W,H的范围在[1,100]，当W,H为0，0的时候结束。

(注 : 保证输入的地图一定包含有出口，以及球球当前所在的位置点)

Output

计算球球最少需要行走多少步才能离开商场，输出“x steps” （x为一个整数，表示最少步数的值）。

Sample Input

5  6<br/>#.###<br/>#.###<br/>#*..#<br/>###.#<br/>....#<br/>##.##<br/>0  0

Sample Output

3 steps

HINT

在Input的输入里，球球往上(/往北)行走两步后，刚好处于商场的出口位置，再往上(/往北)行走一步后走出了商场，因此总共需要3步。


而球球通过另外一个出口离开商场则总共需要8步。
