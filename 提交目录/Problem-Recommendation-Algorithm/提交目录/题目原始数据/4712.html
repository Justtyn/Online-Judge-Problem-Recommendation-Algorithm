Description

最近小明写了一个贪吃蛇的小游戏，但是程序有BUG，常常会控制失灵，贪吃蛇将不受控制一直向前冲去。小明想知道在GameOver之前，贪吃蛇还能走几步。

现在给你控制失灵时的游戏画面，和贪吃蛇头部的坐标以及前进的方向，请你帮小明计算一下贪吃蛇在死之前还能走几步（当他碰到墙或者自己时就认为死了）。

Input

首先，输入一个数字T，( 0 < T <= 10000)，下面将有T组数据。每组数据先给出N，M (0 < N，M <= 20)，代表着游戏的画面的长宽。接下来给出 N行字符串，每行M个字符。#代表墙，_代表空地，$代表蛇，最后将给出蛇头的坐标X，Y。（左上角坐标为0，0）以及方向 Z（用N，E，W，S表示，N为向上， E为向右，W为向左，S为向下）。给出的数据有且仅有一种蛇的形状可以对应。保证地图总被墙包围着。

Output

每组数据输出一行，贪吃蛇还能走几步。
