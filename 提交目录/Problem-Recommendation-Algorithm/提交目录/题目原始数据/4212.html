Problem 4212 --采蝶轩

4212: 采蝶轩

Time Limit: 1 Sec  Memory Limit: 256 MB
Submit: 0  Solved: 0

Description

 采蝶轩面包房采用会员制，每种面包的会员价会比普通价格便宜一些。
        现在给出n (n<=20)种面包的价格和会员价，接下来有m (m<=50)个顾客来购买面包，他可能有会员卡也可能没有会员卡。作为一个收银员，你要告诉每个顾客应付的价格。

Input

  输入数据包含两部分。首先描述了采蝶轩面包的价格。第一行输入一个整数n表示一共有多少种面包(编号0到n-1)，接下来的n行，每行包含两个整数p q分别表示第i号面包的价格和会员价。
        第二部分描述了顾客的行为。接下来一行输入一个整数m表示有m个顾客。接下来的 m行每行描述一个顾客，首先两个数字x r，x表示该顾客是否有会员卡，0表示没有，1表示有，r表示他一共购买了多少个面包。之后跟着r个整数，表示顾客买的面包编号。

Output

对于每个顾客输出他应付的价格。

Sample Input

3<br/>10 8<br/>6 5<br/>8 6<br/>2<br/>0 1 0<br/>1 2 1 0<br/>

Sample Output

10<br/>13<br/>

HINT

Sample说明：

以上示例数据中，一共有3种面包如下：
编号　价格　会员价
0　 　10　 　8
1　 　6　 　 5
2　 　8　 　 6

接着有2个顾客。
第一个顾客没有会员卡，买了1个编号为0的面包，应付10元；
第二个顾客有会员卡买了2个面包，分别是1号和0号面包，应付8+5=13元。
