Description

幸福感是什么?对于农民来说，旅游，是一个足以拥有它的方法。（ dcMoon say :  =. = ）

于是乎农民决定从家所在的城市s出发去旅游。

每次农民从城市a到城市b或者从b到a，农民都能增加w值的幸福感（只要从a到b有幸福感，从b到a也会有相同的幸福感，而且幸福感不会因为城市去过的次数而降低）。但是每次要去一个城市的话，都必须先消耗农民m个金币来准备出发，不够钱就去不成了。

农民从家里只带了n个金币，他想通过这n个金币来旅游来获得最大的幸福感，当然在相同幸福感的情况下金币花得越少越好。假设农民一开始幸福感为0，城市总共t个，标记为1到t。

Input

将有多组测试数据。

对于每一组测试数据，首先用一行给出 t,s,n,m。接下来的多行中，每行包含三个整数 a,b,w. 表示从a到b(同时也是从b到a)能获得w值的幸福感,该三元组输入结束的标志为 a=b=w=0。

所有数据输入的结束标志为 t=s=n=m=0。

Output

对于每一组测试数据，用一行输出参数 i ,j。即农民能获得的最大幸福感，以及其需要花的最少金币。

Sample Input

4 1 1 2<br/>1 2 2<br/>2 3 2<br/>1 3 3<br/>3 4 1<br/>0 0 0<br/>4 1 4 2<br/>1 2 2<br/>2 3 2<br/>1 3 3<br/>0 0 0<br/>0 0 0 0<br/>

Sample Output

0 0<br/>6 4<br/>
