Description

小Y跟小Z在玩游戏。

小Y拿出n个盒子里，并且进行编号1...n。他在某些盒子里放着苹果，让小Z猜哪些盒子里有苹果。（一个盒子最多只能放一个苹果）

小Z会向小Y询问m次，每次小Z会指定某些盒子，问这些盒子里总共有多少个苹果，小Y都会给出一个答案。根据这些信息，小Z要判断出哪些盒子里有苹果。

Input

输入：

多组测试数据。测试数据的开头是一个正整数t，表示测试数据的组数。

每一组测试数据都以n，m开头，分别表示盒子的个数和小Z询问的次数。接下来m行，每一行都会有一个长度为n的01串，表示小Z该次询问指定的盒子，（0表示不指定，1表示指定）。01串后会有一个整数，表示小Y对该次查询的回答。(0 < n <= 20 , 0 < m <= 100)

Output

每组测试数据输出一行。

如果没有一种情况符合，输出"IMPOSSIBLE"。（这表示小Y说谎了）

如果超过一种情况符合，输出"NOT UNIQUE"。

如果只有一种情况符合，则输出代表结果的长度为n的01串，0表示该盒子有苹果，1表示该盒子没有苹果。

Sample Input

1
4 4
1000 1
0110 1
1001 1
0011 1

Sample Output

1010

HINT


有4个盒子，4个查询：


第一个查询：第一个盒子有1个苹果。


第二个查询：第二个盒子和第三个盒子共有1个苹果。


第三个查询：第一个盒子和第四个盒子共有1个苹果。


第四个查询：第三个盒子和第四个盒子共有1个苹果。


根据以上信息，可以确定第一个盒子和第三个盒子有苹果。
