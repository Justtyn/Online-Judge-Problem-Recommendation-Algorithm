Description

 

         112不止有学长，学弟，还有一群萌萌哒（zhong kou wei）的学妹，有猥琐大妈，有阿扎，有打杂少女，有糙妹以及掌握着生杀大权的大姐大YQL。

         这不学长为了显示自己的高大帅气有内涵，多金貌美塞潘安，整天黑她们，这不，终于引起了公愤，YQL决定带头画个圆锥反抗他。

         什么，画圆锥？我们的问题又来了..

        

        

         设有一半径为r，长度为l的圆柱体，平放在深度为2r的水池中（圆柱体的侧面与水面相切）。设圆柱体的比重为p（p>1），现将圆柱体从水中移出水面，问需做多少功？

 

 

Input

         首先输入一个整数T，表示有T组样例

         接下来T行，每行包含三个实数r,l,p表示圆锥的底面半径和高以及圆柱体比重p。（r<100,l<100，p>1）

 

Output

         每组样例输出一行，包含一个实数，表示所做的功，结果保留两位小数。

 

Sample Input

2
10 10 1.2
4.37 1.55 1.6

Sample Output

43982.30
894.02
