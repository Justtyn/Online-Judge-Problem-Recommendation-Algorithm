Description

一台机器，每时只能执行一个任务。现有N个任务，每个任务在这台机器上执行都有个开始时间si和结束时间ti，从时间si开始，到时间ti为止都执行这个任务才算完成了这个任务。问这台机器最多能完成几个任务？

Input

多Case，每Case第一行输入N，接下来的N行，每行两个数，第i+1行表示第i个任务的开始时间和结束时间。0<N<=10^5，其余的都在int32内

Output

每个Case输出一个数，表示最多能完成的任务数

Sample Input

5
1 2
3 5
2 5
5 9
4 6

Sample Output

2
