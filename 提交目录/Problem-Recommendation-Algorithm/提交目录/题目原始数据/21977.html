Description

      英雄联盟里面有一个英雄叫做盲僧-李青，他是一个近距离攻击的英雄，他的被动技能是他每施放一个技能就可以获得一些攻速，持续3次普通攻击的时间，他的Q技能是一个可以远程对敌人造成伤害并且标记敌方单位的技能，再次施放Q技能会使他飞向敌方目标（认为会触发被动技能），W技能可以快速移动到友方单位位置，二段W会有额外效果（认为会触发被动技能），E技能可以在李青身边造成伤害，二段E会有额外效果（认为会触发被动技能），R技能可以使敌方目标被自己击退。

      接下来我想让你们算算，假设刚开始我们与敌人的距离，一段的W技能、R技能产生的距离变化是一样的，一段W只能向背对敌人的方向使用，并且移动到敌方目标的时间等于一次普通攻击的时间，如果在有攻速加成的时间（A）里一直普通攻击，最多能攻击多少次。

      在放技能的时候，立即刷新被动，即被动不会叠加。

Input

多组数据。
每组数据由若干个QWERA组成。（总数不超过50个）
QWER代表技能。
A为规定的普通攻击的时间，且不会在第一位出现。
技能和A会按顺序给出。

Output

输出一行结果表示规定时间里普通攻击的次数。

Sample Input

QQA
QAQ
QAQA
QQQAQA

Sample Output

3
2
5
6
