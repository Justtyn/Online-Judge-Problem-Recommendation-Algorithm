Description

六一儿童节就要到了，晨晨学校组织n位学生去农场摘李子。为了体现同学友好，大 家把摘到的李子集中起来，然后平均分配给学生，剩余的李子就送给老师；另外，为了让老师也更多地分享同学们的快乐，同学们还约定：如果按前面办法分配后老师得到的李子数比每个同学的少，则每位同学再拿一个出来送给老师。

           现在晨晨想知道每位同学最后能收获多少个李子？送给了老师多少个李子？

Input

第一行：一个整数n（1≤n≤200）。

          第二行：n个200以内的正整数，它们之间用一个空格隔开，代表每人摘到的李子数。

Output

第一行：一个整数，代表每位学生最后能收获的李子数。
第二行：一个整数，代表老师最后能收到的李子数。

Sample Input

4
3 5 2 1

Sample Output

2
3
