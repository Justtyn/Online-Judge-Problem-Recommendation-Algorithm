Description

嘉豪是ACM协会的数学小天才，一天他解决了一个问题：M，A，T，H， I，S，F，U，N这9个字母构成的全排列9!中，存在多少种排列，使得在排列中，"MATH"，"IS"，"FUN"各自都不作为顺序且连续字母出现？例如，排列MATHISFUN、INUMATHSF和ISMATHFUN是不合法的，因为分别发生了连续的{MATH...FUN} {MATH} {...MATHFUN}。而{MIUHASFTN}是合法的。聪明的他希望将游戏交给会编程的你来解决，让你来帮他完成这个游戏。

Input

在所有测试数据的第一行，给出一个整数T，代表将有T组数据。

在接下来的T组数据中，每组数据占据一行，由三个字符串组成，字符串之间由一个空格隔开。

单词中不会出现重复字母；所有字母均为大写；单词长度大于1；三个单词的字母总数不超过26.

 

Output

对于每一组测试数据，单独输出一行，为 一个整数P。由于这个排列数N可能很大，所以我们只需要你计算的这个P为所求排列数N模10007之后的结果。

Sample Input

2<br/>MATH IS FUN<br/>ABCDEFGHIJKLMNOPQRST UVW XYZ<br/>

Sample Output

7441<br/>4779<br/>
