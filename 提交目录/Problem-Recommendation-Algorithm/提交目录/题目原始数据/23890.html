Description

考虑两个降序的整数序列X[0..n-1] 和 Y[0..n-1] 并且 X[i] >= X[i+1] Y[i] >= Y[i+1] 而 对于所有的 i, 0 <= i < n - 1. 两个元素之间的距离X[i] 和 Y[j] 由下式给出d(X[i], Y[j]) = j - i 如果 j >= i 并且 Y[j] >= X[i], 否则为0。序列X与序列Y之间的距离被定义为 d(X, Y) = max{d(X[i], Y[j]) | 0 <= i < n, 0 <= j < n}，你可以假设0 < n < 1000.

例如, 对于以下的序列X和Y, 它们的最大的距离为是 i=2 并且 j=7的情况，因为d(X, Y)=d(X[2], Y[7])=5. 

Input

编写一个程序,读若干组整数序列,并分别打印这些序列之间的距离。每一组第一个是X序列，第二是Y序列。你可以假设，序列递减且长度相等.每一行序列之间的数由空格分隔。尝试写一个高效率的程序。

Output

Sample Input

2
9
8 8 4 4 4 3 3 3 1
9 9 8 8 6 5 5 4 3
7
6 5 4 4 4 4 4
3 3 3 3 3 3 3

Sample Output

The maximum distance is 5
 
The maximum distance is 0
