Description

 

         由于今年区域赛的名额很多，教练决定派两名13级的acmer前往千里之外的鞍山参加比赛，但是112的13级学弟学妹可不止两个，于是决定进行选拔赛，最后，三名112神秘人士参加了选拔赛。

         比赛期间，天空乌云密布，112电闪雷鸣，可谓惊天地，泣鬼神。

         赛毕，一切恢复正常，风轻云淡，比赛结果却不被世人得知。

        

 

         比赛方式采取正规的ACM赛制，取若干题目，AC一道计算一次时间，提交错误一次罚时20分钟。AC题数最多且用时较少者获胜。

        

         现在给你3个人的比赛结果，请求出获胜者。

         为了方便，给比赛结果可能如下：

 

 

0      表示该题没有AC

0（-y）   表示该题没有AC，且提交错误y次

         x         表示用时x分AC一道题

x（-y）   表示用时x分AC一道题，且提交错误y次

 

注意：提交错误但没AC不罚时。

Input

         首先输入一个整数T，表示有T组测试样例

         每组测试样例包含4行，第1行输入一个整数n（0<n<6）,表示有n道题目，接下来3行为字符串（含空格），依次表示CJS,LMC,QHJ的比赛结果。

Output

         输出获胜者的首个大写字母。

Sample Input

2
1
99
80(-1)
0
4

10(-2) 20(-1) 50 1
