Description

在输入中，每个点标识一个单一的大写字母(最多为26个). 输入的每一行标识由道路连接点的一对。如果有一行包含“**”表示输入结束。所有的道路都是双向的，即道路AC和CA是相同的。任意一对点之间只有一条公路。
