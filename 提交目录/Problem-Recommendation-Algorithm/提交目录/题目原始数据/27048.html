Description

品种距离proximity.pas/c/cpp

    

FJ的N(1 <= N <= 50,000)头奶牛站成一行，每头牛有一个品种编号。当两头相同品种的牛之间的牛少于K(1 <= K < N)头的时候，这两头牛会发生争吵。

求会发生争吵的牛的最大品种编号。

Input

第一行，2个整数N和K。 
第2..1+N:，每行1个整数，顺序给出每头牛的品种编号（范围是0..1,000,000）。

Output

一个整数，表示会发生争吵的牛的最大品种编号，若没有会争吵的牛，则输出-1。

Sample Input

6 3
7
3
4
2
3
4

Sample Output

4

HINT



【样例说明】




品种3的牛会发生很吵，品种4的牛也会发生争吵，最大编号是4。
