Description

挖孔就是一种钻孔，具体地说，就是钻一个隧道、一口井或者在地球上钻一个洞。随着最近发生的一些事件，如深海地平线石油泄漏和智利矿工的救援，公众开始意识到当前钻孔技术的复杂性。使用地质导向这种技术，钻孔运营商可以垂直地、水平地，甚至在一个倾斜的角度上钻井。
一个钻井计划是在钻孔前准备的，它指定了一系列的路线，代表一个几何数值形状的未来。然而，在钻孔过程中获得了新的信息，是可以更新模型和修改计划的。
你的任务是编写一个程序，通过验证钻孔本身不会相交来验证这个钻井计划的有效性。一个二维的钻井计划是用来代表一个垂直截面的钻孔，这个计划包括一些发生在(0，−1)和移动到(−1，−5)的钻孔。你将在你的程序中编码当前钻井计划并如下图所示：

Input

输入一系列的钻孔命令对。钻孔命令对始于长度是正数的四方向指标之一 (d下降，u上升，l左，r右)。有一个额外的钻孔命令以任意整数q(退出)显示，表示程序应该停止执行。你可以假定输入这样的钻点是不会的：
•不会上升到地面上，
•不会下降到地面以下200个单位以下
•不会偏离原始出发点左侧200个单位，
•不会偏离原始出发点右侧200个单位。

Output

假设钻井已经在图上显示，该程序应继续监控钻孔。正如我们看到的(−1, −5)是您程序上的开始位置。每个命令后,程序必须输出一行新位置的钻孔坐标和对应的信息safe。如果之前的位置没有交叉点或之前的钻孔交叉位置是危险的,输出DANGER，发送自检报告,你的程序必须停止。

Sample Input

d 2
r 1
q 0
