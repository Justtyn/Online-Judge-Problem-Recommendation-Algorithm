Description

评估开始的日期已经越来越近了，全校师生都在积极地准备着，忙得天昏地暗。当然，爱音乐的小wgc也是这样，他最近忙着打扫卫生，按时做作业，当然，也不忘每天坚持练习乐器。正巧，专家中也有不少爱好音乐的，所以就将文艺展示作为考察的一项，为了成功通过评估，学校开始四处寻人给专家表演了，此时会各类乐器的小wgc坐不住了，毛遂自荐，希望能给学校争光，学校毫不犹豫地就答应了。转眼到了文艺展示的当天，小wgc把所有的乐器都抬到了求索广场，等待着专家的考察。专家看见阵容如此庞大，也想考察一下小wgc的功底，就出了一个难题：专家把每个乐器用数字编了号，当专家说出某个数字时，小明就得演奏数字相对应的乐器。问题来了，如果专家让小wgc不间断地演奏各种乐器的话，他可能会累趴在舞台上，所以聪明的你来帮帮他化解这个难题吧。

PS：1对应Guitar    2对应Drum      3对应Ocarina      4对应Flute

         5对应Keyboard  6对应harmonica  7对应Piano        8对应Mouth

Input

在第一行，专家会先给出一个数n（n < 100)，代表专家将会进行n轮访问，然后接下来的n行，专家会给出一个数m（m < 10000)，在之后输入m个数，每个数字之间用空格隔开，这m个数代表着以上乐器的编号。如果m为零或者m个数中有编号为0时，输出applaud。

Output

每输入一个数字，输出对应的乐器，并换行，在输出一轮中的所有乐器后，空一行。

Sample Input

3
4 1 2 3 4
2 5 3
1 8

Sample Output

Guitar
Drum
Ocarina
Flute

Keyboard
Ocarina

Mouth
