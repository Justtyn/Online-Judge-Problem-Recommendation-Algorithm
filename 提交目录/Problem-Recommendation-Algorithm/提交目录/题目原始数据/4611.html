Description

小茗在看了婓哥数列后，对婓哥字符串也很感兴趣


婓哥有一天无聊就给婓哥字符串下了定义 : str[n] = str[n-1] + str[n-2] ( n > 1 )

小茗对婓哥的字符串非常疯狂，如果有人给她两个串，str[0] 和 str[1],她会计算出str[2],str[3],str[4] , str[5]....
例如：
初始值为 str[0] = "ab"; str[1] = "bc";

他会计算出, str[2]="abbc", str[3]="bcabbc" , str[4]="abbcbcabbc" …………;

由于文章很长，ZHP不能把所有的婓哥字符串都记下来， 他只想知道有多少字母出现在婓哥字符串中 ，ZHP问小茗，她就是不肯说，他只好无耐了， Can you help him ?

Input

第一行输入一个整数N，代表有几个测试数据！
然后后面有N个测试数据。

在测试数据中有两个字符串str[0], str[1] 和一个整数 K (0 <= K < 50)都以空格隔开。
字符串最多由30个小写字母组成。

Output

For each case,you should count how many times each letter appears in the Kth Fibonacci String

对于每组测试样例，你需要输出有多少个字符在第K个婓哥字符串中，

例如："X:N".
如果你写的程序可以了，但还有问题，请注意，每个测试样例后面有一个空行。
