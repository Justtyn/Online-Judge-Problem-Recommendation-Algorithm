4481: 超级玛丽

Description

        超级玛丽游戏相信大家都玩过。         现在，碧琪公主又被库巴绑架了( 为什么老被绑架？)，马里奥必须再次起程去解救她。         我们都知道，在很多游戏里，在某些关卡间会有一个小小的考验，通过了考验了便可以得到相应的奖励。这里，马里奥遇到了其中一个考验。如下图所示，马里奥站在一个起始位置上，他的目标通过板间的跳跃，到达星星所在的位置，从而拿到星星（奖励为加一条命）。         每个竖直位置上都会有两块悬浮的板（除起始位置只有一个板外，默认马里奥一开始都站在起始位置上）。如果马里奥站在某一竖直位置上的其中一块板上，那可以跳到这个竖直位置上的另外一块板上（如从Ｆ跳到Ｇ，或Ｇ跳到Ｆ）；也可以跳到相邻前后两个竖直位置上的四个板上（如从Ｅ开始，可以跳到Ｃ或Ｂ或Ｇ或Ｆ上）；但不可以跳到不相邻的竖直位置上的板上（如从Ｆ，不可以直接跳到Ｃ）；当然也可以留在当前所站的板上。（竖直位置间的水平距离不需考虑，马里奥有能力跳过）         但在这里，马里奥也是有他的能力极限的。如果，马里奥的能力数值是３，那么，（相对当前所站的板的高度而言）表示他最高可以跳３的高度，最低可以跳３的深度（如果跳到更低的位置，马里奥会摔死的）。         当然，大部分的这些在关卡间的奖励任务都是有时间限制的。现在马里奥他不想放弃得到一条命的奖励，但他又不知道游戏的时间限制，而只知道用最少的步数到达星星的位置肯定能在游戏的限定时间内完成（否则就很难说了……）。         现在，请你帮他算出最少跳几步，便能到达星星的位置。（时间紧迫，你要尽快为马里奥解决这个问题，因为碧琪公主还在被绑架中…..）

Input

        输入有多组数据。输入的第一行输入一整数Ｋ表示有多少组数据。         接下来Ｋ组数据，每组数据第一行输入一整数n(1<=n<=1000)，表示有多少个竖直位置（不包括出发点的竖直位置）。接下来n行（按输入的顺序，对应图上从左到右位置的顺序），每行两个整数(有可能为负数)，分别表示第i个竖直位置上(从左到右，i从1到 n )两块浮板的高度(同一竖直位置上的两块板高度不会相同)。         最后一行输入三个整数H，B，Y，C，分别表示马里奥出发所站的其始板的高度H，马里奥的能力值Ｂ，星星在第Y个竖直位置上；C= 0，表示星星在第Ｙ个竖直位置上较低的一块板上，反之，C=1则表示星星在较高的一块板上。

Output

        结果输出马里奥最少要跳多少步才能拿到星星，前面输出序数词的缩写，对应表示输出为第几组数据。

Sample Input

2
3
3 5
2 6
1 7
4 3 3 0
3
1 8
4 9
3 6
5 3 1 0

Sample Output

Case 1st: 3
Case 2nd: 5

HINT

如上图所示，马里奥能力值为３的话，应输出５。
因为，尽管从A到Ｃ只要跳一步，但A和Ｃ间距离超出了能力值的范围，不能跳。
所以，从Ａ开始他可以走B->D->F->G->E->C这样一个路线，跳６步。但显然，B->D->F->E->C更快捷，只要跳５步便可到达，因此输出5。
