Description

按要求完成以下操作，请根据实际确定变量的类型。实数使用double类型

 将整数17保存到变量a，将整数6保存到变量b

计算a乘于b的值并保存到变量mul

计算a整除b的值并保存到变量div

计算a除以b的余数并保存到变量mod

计算a实数除以b的实数结果并保存到变量rdiv （提示：1.0*a/b）

将变量b递增10

继续将变量b递增20

将变量a递减3

继续将变量a递减5

输出“a*b=”以及变量mul的值并换行

输出“a/b=”以及变量div的值并换行

输出“a%b=”以及变量mod的值并换行

输出“1.0*a/b=”以及变量rdiv的值并换行

输出“b=”以及变量b的值并换行

输出“a=”以及变量a的值并换行

 提示：

整数和整数的算数运算，结果为整数

如果a，b是整数，则 a/b 执行整除，即保留商的整数部分

实数与整数的混合运算，结果为实数

如果a，b是整数，则1.0 * a / b就是实数结果

注意体会求 余数、整除、实数除、递增、递减。
