Description

最近，有个很出名的电视节目：猫狗大赛。在这个节目中，有一群小猫和小狗为了争夺最受欢迎奖，进行各种各样的表演。每一轮当所有的小猫和小狗表演完之后，观众们会进行投票环节，有一部分宠物会留下来，另一部分会被淘汰。

每个观众每一轮只有1次投票机会，选票中包含自己最喜欢和最厌恶的宠物。另外，每个人要不是一个小猫爱好者，要不就是一个小狗爱好者。所以每个观众的选票中一定会有一只小猫和一只小狗。

主办方发现，如果结果跟某个观众的意愿不一样，这个观众不愿意继续看下去。举个例子：如果某个观众喜欢1号小猫，但1号小猫却淘汰了，或者某个观众讨厌1号小猫，但1号小猫却留下来了。他肯定不会继续看下去。主办方想要尽可能多的观众留下来。

现在请你编写程序，得到留下来的最大人数。

Input

多组测试数据。测试数据的开头是一个正整数t，表示测试数据的组数。

每一组测试数据以3个整数c，d，v开头，分别表示表演的小猫的数量、小狗的数量和观众人数。(0 < c , d <= 100 , 0 <= v <= 500)

接下来有v行，每一行有两张选票。第一张选票代表该观众喜欢的宠物，第二张选票代表该观众厌恶的宠物。每一张选票的都是一个字母和一个编号。如果字母是C，则代表该宠物是小猫，编号在1到c之间。如果字母是D，则代表该宠物是小狗，编号在1到d之间。

Output

每组测试数据输出一行，该行有一个整数，表示能留下来的最大观众人数。

Sample Input

2
1 1 2
C1 D1
D1 C1
1 2 4
C1 D1
C1 D1
C1 D2
D2 C1

Sample Output

1
3

HINT


样例解释：


第一组样例，让C1留下或者D1留下，都能留下一个观众，其他情况都不能留下更多的观众。


第二组样例，让C1留下，D1，D2淘汰，能留下三个观众。
