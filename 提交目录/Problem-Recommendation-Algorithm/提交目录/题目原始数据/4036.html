Description

话说，fwj一直在多校里面打酱油。。（囧 - -|||）于是他想出去旅行一下，放松一下心情（一直打酱油，压力山大的说 = =。。）出去旅行总得收购一点装备。fwj习惯将东西分类收拾在一起，这样比较方便找。所以，他想多买几个背包，每个背包放不同的东西。买背包的时候，fwj发现背包都比较大，每类东西买一个背包就可以装下全部的物品。但是呢，fwj只有W元rmb，买不了很多背包，庆幸的是并不是所有的东西都是必须带上的。于是他把所有想买的东西分类后，标记上一个重要值。怎样才能让带上的东西重要值之和最大呢？这个背包fwj不会呀！！！（T_T。。。）请你来帮帮他吧！

Input

    多组测试数据，读到文件尾结束。对于每组测试数据：
第1行 两个正整数 N，W。W表示fwj的rmb，N表示fwj想买物品共有N类。
第2行~N+1行 每行开始有两个正整数Mi，Ni。表示买该类物品的背包价格为Mi，该 类物共有Ni个，接着Ni对数pi，vi表示这件物品的价格为pi，重要值为vi。
1<=N<=50
1<=W<=100000
1<=Mi<=1000
1<=Ni<=10
1<=pi<=100
1<=vi<=1000000

Output

每组测试数据输出一个正整数表示重要值之和的最大值。

Sample Input

3 800
300 2 30 50 25 80
600 1 50 130
400 3 40 70 30 40 35 60

Sample Output

210
