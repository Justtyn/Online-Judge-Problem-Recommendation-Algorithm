4742: 签到

Submit: 0  Solved: 0

Description

        参加许多活动时都会有签到这种神奇的事情发生。小罗非常厌烦签到，但是没有办法，不能拒签。有一次n个人的活动，小罗要签到了，他看了下中哥来了还没有签到，于是他把中哥的学号写上去；中哥看到了觉得很好玩，又找了一个来了但还没有签到的人的学号写上去...如果大家都很配合的话，来了的人都能在签到纸上有自己的学号，而且每个人都只签一个学号。

 活动结束后，有m个人想知道谁签了他的学号，你能告诉他吗？

Input

 单组测试数据。
 首先一对n、m(0<m<=n<=50)。
 然后n行，每行两个数
 a b 表示编号为a的同学替编号为b的同学签到
 所有人的学号都不同，(0<a,b<=100000000)
 接下来又是m行，每行包含一个有效的学号c。

Output

输出m行，每行输出签学号c的人的真正学号。

Sample Input

3 3<br/>1 200<br/>200 1<br/>5 4<br/>1<br/>200<br/>4<br/>

Sample Output

200<br/>1<br/>5<br/>
