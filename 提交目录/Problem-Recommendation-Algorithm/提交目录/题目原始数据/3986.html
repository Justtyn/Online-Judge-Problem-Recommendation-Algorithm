Description

话说众人终于要汇合了，这时候，Rabbit大神特地从家里带来了好多兔肉干-_-|||

现在知道准备汇合的两队人数分别为p和q，Rabbit大神特地给每一支队伍准备了一块，当然，这一块一个队得分着吃，但是如果分的不平均的话大家可能会吵起来影响救援宣姐的进度，所以，有个问题摆在了Rabbit大神面前。首先，兔肉干需要提前切好，等到人来了的时候就可以马上吃了，但是，Rabbit大神又不知道这两队哪一队会先到，而且时间只够切一块兔肉干，所以这块兔肉干必须保证两支队中无论哪支队过来都有办法分。还有个问题就是这块兔肉干必须平均分给队伍里的每个人，而且兔肉干必须分完。所以，Rabbit大神不知道这块蛋糕该怎么切才符合上述要求，输出最少要把兔肉干分成几块，具体看Hint。

Input

多测试数据，每个测试数据输入两个数p，q，0<p,q<10^9

Output

一个数，为最少的块数。

Sample Input

2 3

Sample Output

4

HINT


将兔肉干分为四块，即1/3,1/3,1/6,1/6，如果是2个人的队伍来的话可以一人分1/3+1/6=1/2块，如果是3个人的队伍来的话可以一人分1/3，最后一个人分到的是1/6+1/6=1/3，允许出现一个人分到好多块的情况，但是总量必须与其他人一样多，所以样例输出4块。
