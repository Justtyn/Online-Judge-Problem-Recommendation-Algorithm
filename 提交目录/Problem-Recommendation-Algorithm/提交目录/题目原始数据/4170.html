Description

相信打过DOTA的朋友都知道炼金术士比一般的英雄打钱要快，因为他有一个技能叫“地精的贪婪”，这个技能可以使他杀怪可以得到额外的金钱，我们来看看技能介绍：

 

 

炼金术士在杀死敌方单位时能得到额外的金钱奖励，而在18秒内连续杀死单位更可以提高2的金钱奖励额度，该技能获取额外金钱上限为18。

等级1:每杀死一个敌方单位得到额外的2的金钱奖励。

等级2:每杀死一个敌方单位得到额外的4的金钱奖励。

等级3:每杀死一个敌方单位得到额外的6的金钱奖励。

等级4:每杀死一个敌方单位得到额外的8的金钱奖励。

另外，游戏开始后，每过一秒钟，系统会发放1的金钱给玩家。

我们的目的是，给定一些杀死怪物的时刻和得到的金钱，求杀死最后一个怪物时，炼金所拥有的金钱，游戏从第0秒开始。

Input

有多组测试案例，每个测试案例的第一行一个整数n，m（0 < n ≤10000, 0 ≤ m ≤ 4），n代表杀怪的数目，m代表技能的等级；接下来的n行，每行两个整数ai，bi(0 ≤ ai ≤ 50000，0 ≤ bi ≤ 100)，代表第ai秒时杀死价值bi金钱的怪物；输入0 0表示结束。

Output

对于每组测试案例，输出炼金术士杀死最后一个怪物时拥有的金钱，占一行。

Sample Input

2 1
1 50
2 50
3 1
1 50
2 50
20 50
0 0

Sample Output

108
180

HINT

第一组样例：两只怪提供100金钱，技能两只怪加4金钱，打第二只时与第一只间隔在18秒内，再加2金钱，游戏时间2秒，提供2金钱，计算如下：


（50*2）+（2*2）+2+2*1=108
