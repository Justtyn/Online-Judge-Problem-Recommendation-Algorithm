Description

你是空调机械公司的技术员。不幸的是，当你去为顾客安装空调管道时发现

材 料 并 不 充 足 。 你 只 有 六 段 管 道 ， 每 段 管 道 都 是 一 样 的 “ 肘 状 物 ”。


你只能在一个限定的空间内装载管道：一个边长为单位长度的整数倍的长方

体。因此，可以认为它是由一系列单位立方块所组成的。每个肘状物恰好占据四

个立方块，如 Figure1 所示。每个单位小立方体只能被一个肘状物所占据。每个

肘状物恰好有两个开口，如 Figure1 所示。在不超过给定空间的前提下，你可以

把多个肘状物衔接起来以构成更长的管道。Figure2 举例说明了一种把两个管道

衔接的方案。你的任务是把水流入口和水流出口连接起来。入口和出口都位于给

定空间的外表面上，如 Figure3 所示。为了降低成本，你需要用尽量少的肘状物

来完成这一任务。

 

 

Input

输入第一行为一个正整数 T，表示数据组数。本题有若干组输入数据，每组

数据仅占一行，包含 11 个元素。格式如下：

首先是 3 个整数（Xmax，Ymax，和 Zmax），分别表示空间在 x、y、z 方向上

的范围。因此，每个单位小方块的坐标均可以表示为（x，y，z），其中 1 ≤ x ≤

Xmax, 1 ≤ y ≤ Ymax, and 1 ≤ z ≤ Zmax。Xmax，Ymax 和 Zmax 均为不大于

20 的正整数。接下来是 3 个正整数，表示水流入口的坐标。接下来是一个长度

为 2 的字符串表示水流流入的方向，可能的取值有：+x，-x，+y，-y，+z，-z。

水流是从单位小方块的某个面流入的，例如+y 表示水流从该方块的 y 轴反方向

的面流入。然后有 3 个正整数，表示水流出口的坐标。接下来是一个长度为 2

的字符串表示水流流出的方向，可能的取值同上。水流从某个小方块的某个面流

出，例如+y 表示水流从该方块的 y 轴 正方向的面流出。

Output

对于每组数据，输出数据组号（从 1 开始），接下来一个整数表示完成任务

最少需要多少个肘状物。若无法用 6 个以内的肘状物完成任务，则输出

Impossible。具体格式见样例


Sample Input

2
5 4 3 3 1 1 +z 5 4 3 +x
5 4 3 3 1 1 +z 1 2 3 -x

Sample Output

Case 1: 2
Case 2: Impossible

HINT


对于 40%的数据，Xmax，Ymax，Zmax 中至少有一个为 1；



对于 100%的数据，Xmax，Ymax，Zmax 均不大于 20。
