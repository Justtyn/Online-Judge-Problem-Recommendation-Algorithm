Description

　　巴斯光年是妮妮的好朋友。最近，妮妮将从巴斯那里收到很多宇宙的新科技产品，作为巴斯给他和他那些地球好友的圣诞礼物。 　　礼物经由宇宙速递送达到地球。当然，在妮妮签收的时候，她需要确定收到的礼物数量是否和巴斯寄出的礼物数量一样。但是宇宙速递的习惯跟地球上有点不一样。她发现签收的单子上只写着一串字符，用来表示不同的礼物盒和各种各样的礼物。一共有四种礼品盒，分别表示为”< >” , ”{ }” , ”[ ]” 或 ”( )”。 剩下的其他所有字符则表示不同的礼物。 　　宇宙礼品盒相当神奇，它可以装任意多的礼物，也可以嵌套地装任意多的其它盒子。 　　所以，妮妮需要对比收到的礼物盒数量，是否跟字符串反应的信息一样。但她对字符并不敏感，而且由于巴斯送出了好多的礼物，用了好多的礼物盒，所以妮妮看了一会儿就感到头晕了。 　　于是，妮妮急需你的帮助。对于每段字符串，请帮她算出来所描述的礼物盒从外面看来，总共能看到有多少个盒子？

Input

　　输入有多组数据。每组数据输入一行字符串，表示妮妮收到的礼物盒的信息。当输入字符串为”END”的时候，表示输入结束。字符串长度不超过300个字符。　　”< >” , ”{ }” , ”[ ]” ，”( )”分别表示四种礼物盒，其他任意字符（可从键盘上直接输入的任意字符）表示不同的礼物。　　所有输入的数据，保证括号匹配，不会出现有“<[>]”的情况。

Output

　　每组数据输出一行，一行包括一个整数，表示从外面看，究竟能看到有多少个礼物盒。

Sample Input

{{*}*}<br/>{*}[%](#&)<br/>END<br/>

Sample Output

1<br/>3<br/>
