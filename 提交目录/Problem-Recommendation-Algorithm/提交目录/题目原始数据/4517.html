4517: 直线上的点

Description

Petter最近在研究个简单的问题。 在一条直线上， 有N个点，从中选3个点。 他想知道有多少种方式能让选择的点的最大距离不超过D.

注意：N个点是严格递增的。 也就是说没有重合的点.

Input

第一行有一个T表示， 有T组测试数据。

下一行有一个N (N <= 100000) 和 D (在int范围内)，分别表示点的数量和选择的点的最大距离D。

接着一行有N个整数A[i] (在int范围内) , 0 < = i < N; 这保证A0 < A1 < A2 < …… < An-1.

Output

对每组测试数据输出最多能有多少种选择方式。

Sample Input

3
4 3
1 2 3 4
1 2
1
5 19
1 10 20 30 50

Sample Output

4
0
1

HINT

第一组测试数据有4种取法 {1, 2, 3} , {1, 2, 4}, {1, 3, 4}, {2, 3, 4};


第二组测试数据不足3个点，0组取法。


第三组测试数据只有一组取法 {1, 10, 20}
