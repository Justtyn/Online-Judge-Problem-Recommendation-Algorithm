Description

有一种卡片游戏，每个玩家一开始都有13张牌。每个游戏者都有一个初始点数，是根据他们手中牌的点数决定的。每张牌的点数是不一样的，A为4点，K为3点，Q为2点，J为1点，其他牌（2,3,4,5,6,7,8,9,10）为0点。有4种花色梅花（C），方块（D），红桃（H），黑桃（S）,花色不分级别.但是花色有另外的点数规则：空位（a void）3点,单独（a ingleton）2点,对子（a doubleton）1点。“空位”表示该种花色一张都没有（例如，一个游戏者已经没有黑桃）。 “单独”表示某种花色只有一张(例如某个游戏者只有一张方块).“对子”的意思是某种花色只有两张.

写一个程序，用字符串的形式读入这13张卡片，然后输出他的总点数. 牌按照递增顺序存放而且没有相同的牌。

输出按照表格形式输出加分。
