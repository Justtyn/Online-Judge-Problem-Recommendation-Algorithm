在地下城与勇士中，枪手有两种技能：

 

枪手技能介绍：

 

狙击步枪：单体攻击，每次发出1颗子弹，被击中的怪物减少A点生命值，消耗玩家B点魔法值。

 

激光步枪：全体攻击，每次发出1束光线，所有被击中的怪物各自减少C点生命值, 消耗玩家D点魔法值

 

假设现在有N只怪物站成一排，已知它们各自的生命值，请通过上面的两种技能，消耗最少的魔法值，消灭全部怪物。

注意：当某个怪物生命值小于等于0时，代表该怪物已被消灭。

狙击步枪可以选择攻击任意一个生命值大于0的怪物，不一定是站在最前面的一个。

激光步枪每次发起攻击都会击中全部生命值大于0的怪物。

Input

第一行输入一个整数T，代表有T组数据。

接下来的每组数据中占三行，分别是：

第一行输入一个整数N (0 < N <= 100)，代表有N只怪物；

第二行输入N个正整数，每个数不超过1000，代表N只怪物的生命值；

第三行输入四个整数A( 0 < A < 100 ), B( 0 < B < 100 ), C( 0 < C < 100 ), D( 0 < D < 100 )。

Output

每组数据输出一行，包含一个整数，代表消灭所有怪物最少需要消耗的魔法值。

Sample Input

2<br/>2<br/>100 100<br/>100 10 100 30<br/>2<br/>100 1<br/>99 2 1 2<br/>

Sample Output

20<br/>4<br/>
