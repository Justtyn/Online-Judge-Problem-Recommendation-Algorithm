21771: 多少条最短路

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 1  Solved: 1

Description

给定一个包含 v 个点和 e 条带权有向边的图 G 以及其中的点 s 与点 t，求从点 s 到点 t 的最短路有多少条。

Input

输入包含多组测试数据，以 EOF 结束。 每组数据第一行包括四个整数 v，e，s，t。接下来 e 行每行描述一条边，包含三个整数 a，b，w 分别表示每条边的起点编号、终点编号、边权。顶点编号为 0 至 v-1。

Output

每组数据输出最短路的条数。若不可达则输出 0。

0 < v < 1000

0 <= s, t, a, b < v

0 <= w < 2^31


Sample Input

5 5 0 4
0 1 1
0 2 1
1 3 1
2 3 1
3 4 1
5 3 0 4
0 1 1
0 2 1
2 3 1

Sample Output

2
0
