Problem 4419 --庙宇逃亡

4419: 庙宇逃亡

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 0  Solved: 0

Description

最近Bob喜欢上玩这游戏，于是也想简单的模拟下这游戏。规定每秒走一步，只要不掉如河里，就能继续在桥上奔跑。掉进河里，游戏结束。分数就是秒数。起点为（0，0）， 开始为水平向右。

Input

测试数据有若干组。第一行有两个整数，分别为N, M（0 < N , M <= 14).代表N行M列。接下来有N行，每行有M个字符.   只会出现两种字符，'.'代表能桥， 'x'代表河。然后有两个整数S,K( 0 < = K ,S < 50).代表S秒后游戏结束，接着有K行。每行有一个整数P和一个字符a. P代表第P秒末将运动方向变为a. a 可能为 's', 'x', 'z', 'y'，分别代表上，下，左，右。如果N, M 都为0时，程序结束。

Output

每组测试数据输出一行，第一行：如果操作完了掉入河里，则输出"Game over! x y #", 如果仍然在桥上，则输出所在坐标"You win! x y #", x,y代表最后在桥上的坐标， #代表获得的分数。

Sample Input

6 4
....
.x..
.xxx  
.x..
.x..
....
8 4
3 x
4 z
5 s
6 z
0 0

Sample Output

You win!
0 0 8
