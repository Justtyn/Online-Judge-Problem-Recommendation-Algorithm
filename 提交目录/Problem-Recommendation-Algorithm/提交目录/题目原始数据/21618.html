Description

Alice and Fat Brother are playing a game in a graph. This graph is a rooted tree at the beginning and then they take turns to take out some exist vertices. Each move you can choose a exist vertex and take it out of this game as well as its ancestors. The game ends when all the vertices are taken and the one whom takes the last vertex is the winner. Given a rooted tree, your task is to calculate who the winner is. Suppose that both Alice and Fat Brother are smart, they can always find the best strategy in each move and Alice always goes first.


Input

The first line contains only one integer T, which is the number of test cases.
Each test case starts with an integer n, which is the size of the rooted tree.

Then goes n-1 integers s2, s3, …, sn, means that the father of vertex x is sx, you can assume that vertex 1 is always the root.

Output

For each test case, output the case number first. If Alice can win this game, output the number of ways Alice can choose to win for the first move, otherwise just output 0.

Sample Input

4
2
1
3
1 1
4
1 1 1
3
1 2

Sample Output

Case 1: 1
Case 2: 1
Case 3: 3
Case 4: 1
