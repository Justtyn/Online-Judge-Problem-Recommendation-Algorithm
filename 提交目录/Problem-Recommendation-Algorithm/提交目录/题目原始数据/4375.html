4375: C. 天游开车

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 0  Solved: 0

Description

在一个风和日丽的日子，没有驾照的天游开着他的宝牛去兜风了（请勿模仿，此乃无证程序员的行为）。当天游正以V1（m/s）的速度行驶在路上时，发现他的正前方L（m）处的一辆烂脖正以V2（m/s）速度同向行驶。于是为了安全起见，天游决定制动（可认为是匀变速运动），但是制动的话天游的宝牛还要行驶L0（m）才能停下来。问天游的宝牛是否与前方的烂脖相撞，如果不撞，请求出由制动开始，两车的最近距离是多少（若恰好距离为0，即安全）？

Input

第一行输入数字N，表示有N组测试数据，N <= 30；

每组测试数据有一行，4个正整数，分别是，V1，V2，L，L0，分别表示天游的宝牛，前方的烂脖，两车的距离和宝牛制动需要的距离。0 < V1，V2 < 500，0 < L，L0 < 2000。

Output

针对每组测试数据输出一行：

如果相撞输出"OMG!"

如果平安无事输出两车的最近距离（保留两位小数）。

Sample Input

1
20 8 600 1000

Sample Output

240.00
