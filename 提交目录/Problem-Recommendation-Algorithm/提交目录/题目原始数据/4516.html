Description

词语接龙这个游戏相信大家都玩过，规则就是前面一个词语的最后一个字母要跟后面一个词语的首字母要相同，比如abc和cdf就能接上了，而abc和def不能，因为它们不符合上述要求；HJY很喜欢玩这个游戏，因为它既能记忆单词，又能娱乐；但是HJY是个充满好奇的同学，他希望在给出的n个单词当中找出若干个单词，像词语接龙那样接起来凑成一个终极的单词，组成终极单词的前后顺序和输入的前后顺序一致。并且那个终极单词的首字母和尾字母要相同；现在，HJY的问题来了，请你帮他找出终极单词中长度最长那个。

Input

第一行为一个整数T，代表有T组测试数据。
随后每组测试数据的第一行为一个整数N，随后N行，每行一个单词；(n <= 100000，每个单词长度不超过10，只含小写字母)

Output

每组测试数据输出一行，即终极单词最长长度。

Sample Input

3
3
ab
acb
ba
4
good
abc
who
zjk
3
c
ab
def

Sample Output

5
0
1

HINT

若一个单词的首字母和尾字母相同，它也可以单独构成一个终极单词；

输入量大，请使用scanf读入
