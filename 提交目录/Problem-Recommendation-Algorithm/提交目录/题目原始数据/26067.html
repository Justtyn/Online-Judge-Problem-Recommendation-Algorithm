Description

吴邪想着，十年之约已到，便动身再寻青铜门。

风，雪，长白山。

青铜门开。

寂静无声。

黑暗中有些许微光。

小哥……吴邪晃了晃手电，果然，却未得到回应。

角落，那蓝色帽衫的人竟容颜未变，对着墙却不知做着什么。

吴邪凑近，惊讶的看着小哥在训练着几个蘑菇精（……三叔的梗，不说什么了）

看了一会，吴邪发现，小哥有两种操作：

g  x：给x蘑菇精一个蘑菇。

s  x y：让x和y蘑菇精交换他（她？它？）们所有的蘑菇。

几个操作为一组，一直循环。

——小哥，你还要这样做多少次……

——m。

吴邪急于知道m次后的结果，好带小哥回家。

他灵机一动，想起了这次带了笔记本电脑。

于是……

他给你发了邮件……

Input

输入有多组数据

第一行是数据数目T

每一组数据：

第一行三个数n，m，k

n为蘑菇精总数，m为循环次数，k为每次循环的操作数。

接下来k行描述一组操作，每次操作描述如题。

Output

每组数据一行，为m次循环后每个蘑菇精拥有的的蘑菇个数，用空格隔开。

Sample Input

1
3 1 5
g 1
g 2
g 1
s 1 3
g 2

Sample Output

0 2 2
