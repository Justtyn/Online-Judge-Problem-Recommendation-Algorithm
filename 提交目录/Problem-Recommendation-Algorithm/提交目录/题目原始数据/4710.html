Description

大富翁是一款老少咸宜的游戏，棋盘上有N个格子，从0开始依次编号到N-1。玩家从编号为0的格子出发，每次掷色子决定初始走的步数，然后从当前位置开始往前走，每步走一格。到达目标格后，如果目标格的标记为0，则该轮移动完毕；如果该目标格的标记为一个正数，则再向前移动对应的步数；如果标记为负数，则向后倒退对应的步数。有可能出现连锁反应。

注意如果到达编号为N-1的终点格时，还有步数没走完，则向后倒退剩余步数，该轮移动完毕，下次掷出点数再依照先前规则前进。

    现在给出棋盘的情况，以及每次掷色子的点数，问玩家最后所在的格子的编号。棋盘的布局保证合理：0号格和N-1号格的标记为0，不会出现让玩家后退到棋盘起点格之前的标记，但可能出现让玩家前进到棋盘终点格之后的标记，此时仍然要按照前述规则向反方向走剩余的步数。

Input

首先有一个整数T，代表有T组数据。

随后每组数据占2行，

第一行输入大富翁的棋盘格子编号，首先有一个N，6 < N <= 100，代表一共有N个格子，注意编号为0 – N-1，随后有N个合理的整数，依次为从0开始的每个格子的标记。

第二行输入每次应该走的步数即掷色子的结果，每个结果都是1 – 6之间的一个整数，最后-1表示掷色子结束，-1不处理。 输入数据保证不会出现到达终点后继续掷出点数的情况。

Output

针对每组数据输出一行，为最后所在格子的编号。

Sample Input

2<br/>8 0 4 3 2 -1 2 -3 0<br/>1 -1<br/>9 0 1 1 0 -3 4 1 2 0<br/>1 3 -1<br/>

Sample Output

7<br/>7<br/>

HINT

第一组数据只扔了一次色子，点数为1。则第一次玩家走到格子1，标记为4，再走4步，到达格子5，标记为2，继续走两步，到达终点格子7。


第二组数据第一次的点数为1，走到格子1，标记为1，再向前走1格，到达格子2，标记为1，前进到格子3，标记为0，此轮结束。第二子色子点数为3，玩家从格子3前进到格子6，标记为1，再走一步，到达格子7，标记为2，还要走2步，由于不是恰好到达终点，则到终点后还要向后退一步，回到格子7，此轮结束。输出答案7。
