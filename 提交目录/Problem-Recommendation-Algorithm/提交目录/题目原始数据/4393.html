Description

这天，小Y在用苹果自娱自乐。

他手上有m类苹果。小Y要用n个苹果排成一个队列，但该队列中不能有超过连续k个同类苹果出现。 

小Y想知道，有多少种排列方法。

Input

多组测试数据。测试数据的开头是一个正整数t，表示测试数据的组数。

每一组测试数据有三个整数n，m，k，分别表示苹果队列的长度，苹果的种类和苹果队列中允许出现的最大连续同类数。(0 < n < 263，0 < m <= 6，0 < k <= 6)

Output

每组测试数据输出一行，该行有一个整数，排列的方法数。由于结果非常大，输出方法数模1,000,000,007的值。

Sample Input

1
4 2 3

Sample Output

14
