Description

我们学校占地面积5000余亩，总规划建筑面积100万平方米，教学楼、宿舍楼更是数不胜数。在这么大的一个学校上学，出行就是一个很大的问题，虽说有电瓶车，但是天天那条排队长龙，估计你也没有多少机会坐上吧。Lcc是路痴而且还喜欢睡懒觉，因为他不熟悉学校的环境，所以经常走了很多的冤枉路，上早课也是经常迟到。现在，他想请你帮忙编写一个程序让他知道到达目的地的最短路程是多少。

Input

输入有多组数据。每组数据首行是整数N（0<=N<=15）,代表建筑物的总数，M（0 <= M <= N），表示提问的次数。当N=0和M=0时，表示测试结束。

接下来有N*N的矩阵，代表建筑物之间的距离，每个数字都不超过100，如果是-1，代表两建筑物之间没有道路可以行走。

接下来M行，每行有两个数字是s，e（1 <= s,e <= N），s表示lcc所在地，e表示他想去的目的地。

Output

对于每组测试数据首先输出

Case # ：（#代表第几组测试）

对于每组s，e，输出

From s to e :

Length : k(k为s到e的距离)

若不连通：

Length : No!

每组后面都有空行。

（详细见样例输出）
