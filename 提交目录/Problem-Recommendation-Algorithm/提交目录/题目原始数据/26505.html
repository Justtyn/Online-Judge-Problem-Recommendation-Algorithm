Description

　　炮姐最近研发出一种新的电磁炮，这种电磁炮的轨迹为抛物线，并且可以在电磁炮发出后未击中目标之间的某个时间点改变电磁炮的轨迹．

　　如图所示：LYB刚刚抢劫一家银行并躲在Ｂ点，在Ｂ点和Building1之间有一座Building2．炮姐站在Building1的楼顶，她可以发出轨迹方程为(y - y0) = -ax2的电磁炮，并且在电磁炮未到达目标的时候可以有一次改变电磁炮轨迹的机会，如果选择改变则轨迹方程更改为(y - y1) = -b(x - x1)2. 现在给出方程的参数a和b和一些距离参数，那么炮姐能否发射新研发的电磁炮准确无误的击中Ｂ点的LYB，并且不会击到Building2?

Input

不超过１０组测试数据，以文件结尾结束．

每组测试数据包含六个数h1, h2, d1, d2, a, b. h1 是Building1的高度，h2 是Building2的高度，d1 是Building1到Building2之间的Ｘ坐标的距离．d2 是点Ｂ到Building1之间的Ｘ坐标的距离．四个数范围满足[0, 1000] , 并且d1 < d2. a 和 b 也满足在(0, 1000]区间内.

Output

如果炮姐能击中LYB，输出"Yes",否则输出"No".

Sample Input

25 1 6 7 1 1
4 3 1 2 1 1

Sample Output

Yes
Yes
