Description

　　Suby 最近在学线性代数，老师讲了很多关于矩阵的内容。　　矩阵的知识实在是太多了，光是特殊的矩阵就有许多种，像对角矩阵、上三角矩阵、单位矩阵、范德蒙矩阵等等。　　Suby对这些东西十分讨厌，在老师讲课的时候，他开始自己研究起矩阵来。　　一天，他发现了一种有趣的蛇形矩阵。　　矩阵中的元素满足如图箭头所指的1，2，3，...，n×(n-1)，n×n，的蛇形排列的关系　　现在，给你的任务就是用程序来输出一个n阶的蛇形矩阵。

Input

　　输入一个N(1≤N≤80)，代表要求输出蛇形矩阵的阶数。输入数据有多组，输入以EOF结束。

Output

　　对于每个输入，输出对应的N阶蛇形矩阵，矩阵每行的元素之间以空格隔开。每个CASE的输出之间用空行隔开。
