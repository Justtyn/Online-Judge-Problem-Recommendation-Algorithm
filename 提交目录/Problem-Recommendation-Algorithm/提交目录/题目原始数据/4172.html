Description

又见喜人的 A+B，不过，正如题目所述，此A+B有点二。

 

       昨天光棍节寂寞的小煜实在很无聊，不停地切a+b这种水题，不过后来，他做腻了《人见人爱的A+B》这种没意思的题目，于是，创新出一种全新的运算，即《@，# 运算》。

       这两个运算符有比较特殊的含义（具体操作请看样例解析）：

 

       @符号：将两个数的奇数位数字相减（每位上的数字大减小），偶数位的数字取较大值（每位上的数字比较，取最大值）。

 

       #符号： 将两个数的偶数位数字相减（每位上的数字大减小），奇数位的数字取较大值（每位上的数字比较，取最大值）。

 

       特别提示：例如123456789，个位为第1位，是奇数位，数字为9；十位为第2位，是偶数位，数字为8；后面以此类推。

      

       输入一个式子后，请按照这个规则计算后，输出答案。式子均为从左到右运算。

Input

第一个输入为一个正整数T，表示后面有T组测试数据。后面T组测试数据每组有一个正整数N，表示该式子由N个数字组成（数值不超过int范围），并且，每两个数字之间间隔一个运算符（@或#），总共N-1个运算符。

Output

根据输入的式子，经过上述定义的@，#运算后，输出计算结果，输出占一行。

Sample Input

2
4
123456789 @ 8765432 # 121212121 @ 111111111
3
111111 @ 111111 # 101010

Sample Output

14504266
0
