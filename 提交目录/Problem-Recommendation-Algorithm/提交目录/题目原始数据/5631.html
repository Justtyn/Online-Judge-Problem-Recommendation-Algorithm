Description

      跑马圈地是指清朝初期，八旗子弟入关后骑马以圈定土地归属的现象。

清朝起初在关外，土地很贫瘠，生活很艰辛。入关后，发现关内的土地大到他们不敢想象，于是就通过八旗贵族骑着马，在一定的时间内能跑多少就圈定多少土地归他的方式来确定明朝亡后遗留下的大量的皇室土地，继而蔓延到平民百姓的土地身上。后来被康熙制止了这种跑马圈地的行为。

       清初跑马圈地时期，获土地最多的是顺治时期的“摄政王”多尔衮和康熙初年的“顾命大臣”螯拜

方法就是；骑马跑多远的距离多大地就归你。1644年，满清打败李自成入关后不久，1645年（顺治二年）正式颁行圈地令,将所圈占的土地分给东来的八旗将士，开始了规模浩大的跑马圈地运动，把原来属于汉族农民的土地霸占分给八旗贵族（造成大批失地的下岗职工）。失去土地的农民为维持生计，又反过来被迫为满清八旗贵族当包衣（类似于农奴）耕种土地。后清朝政府改变它的经营方式,组织农庄。庄分大庄(地420-720亩)、半庄(地240-360亩),把土地拨给八旗王（诸如螯拜之类）、公、宗室,由内务府统一管理,将所收地租分给王公宗室。官庄采用租佃出租给农民,改变过去以包衣为主要劳动力的经营办法。官庄设庄头,直接管理农民,征收地租。庄头是二地主,是恶霸,《红楼梦》里写乌庄头,就是这种二地主的艺术形象,它写的乌庄头交租,基本上反映了官庄的经营方式。

       跑马圈地是种掠夺土地的行为，把骑马跑过的土地划入自己的封地，扩大自身势力范围

       话说,话说Miaowu和A,B,C三人来到了一个新大陆,由于Miaowu是老大,所以它决定其他3人由跑马圈地来决定哪些土地是属于哪些人的! 为了让题目变得简单,我们假设每个人圈的地都是平行于平面坐标轴的,并且每个人圈得的地盘是一个矩形.当然因为大家都不知道哪些地已经被人圈过了,所以可能会出现同一块地被多人圈的情况,这是一种很不好的情况,几个人可能会为此大动干戈,因此为了避免几个人争地,Miaowu决定:1.假如某一块地只有A圈,那A得到这块地,其他人同理  2.假如一块地被A,B同时圈中了,那么这块地归C,这样好像不会产生什么纠纷,同类型的情况同理. 3.假如A,B,C同时圈中了某块地,这就麻烦了,但聪明的Miaowu还是决定,如果这种情况发生了,那这块地就属于Miaowu.

      Miaowu是一个很想知道结果的人,所以想请你告诉他每个人分地的面积.

Input

                   输入有多组数据,一直到文件末尾.每组数据的第一行一个整数n,表示一共圈出了n块地,接下来n行,每一行的格式是name  x1  y1  x2  y2.  (name=”A” or  “B”  or  “C”)表示哪个人圈的这块地,(x1,y1),(x2,y2)分别是这块地的左下角坐标和右上角坐标.

数据范围:  1<= n <= 10000,  0<= x1<=x2<= 10^9,  0<=y1<=y2<=10^9. x1,y1,x2,y2均为整数

Output

对于每组数据输出5行,第一行输出Case #k:,k为第k组数据,  第2行是Miaowu得到的面积,第3行是A得到的面积,第4行是B得到的面积,第5行是C得到的面积


或许,你需要用 long long (%lld);
