Description

一年一度的集训时间又到了，小胖和洋姐都励志要在暑假怒刷rating。不过不是codeforces，而是lol！但在得知集训室不准打游戏后，他们找到了一种新玩法。发明了lol纸牌游戏，类似游戏王的一种卡牌游戏。

 

小胖有n张卡，每张卡都有两个属性：状态（攻击或防守）和power。洋姐有m张卡，每个卡也有这两个属性。

 

现在是小胖的战斗阶段，他可以执行下列操作多次：

 

1 选择他的一张卡X。此卡之前不能被选择过。

2 如果洋姐当前没有卡活着（初始时所有卡都是活着的），他受到的伤害等于（X的power）。否则，小胖需要选择一张洋姐的卡Y：

3 如果Y的状态是攻击，那么小胖必须使（X的power） ≥ （Y的power）。攻击后，卡Y死亡，洋姐得到的伤害等于（X的power）-（Y的power）。

4 如果Y的状态是防御，那么必须使（X的power） > （Y的power）。这种攻击后，卡Y死亡，但洋姐不受到伤害。

 

现在小胖和洋姐开启了一盘游戏，因为小胖是大神，所以他能看穿洋姐的布置，即知道他会把每张卡设置成什么状态，且因为小胖让着洋姐，所以他的卡片比洋姐少。但小胖很嚣张，所以他把所有卡都置为攻击状态。现在，他需要考虑怎么把他对洋姐的伤害达到最大，但他最近撸多了，所以智商拙计，于是需要你帮帮他。

Input

第一行是整数t，代表测试数据组数。

第二行是两个整数n和m，（1<= n < m <= 1000),分别是小胖和洋姐持有的卡数。

接下来的n行，每行分别表示小胖一张卡的power值，

再接下来m行，每行表示洋姐卡的状态和power值。ATK表示攻击状态，DEF表示防守。（0<=power<=10000)

Output

输出一个整数，洋姐可能受到的最大伤害。
