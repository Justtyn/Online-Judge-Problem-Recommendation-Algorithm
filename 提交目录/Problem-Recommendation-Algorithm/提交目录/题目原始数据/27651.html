27651: 一摞盘子

Description

张静静在餐馆给人打工洗盘子。他把盘子摞成一摞，他不断地把最顶上的盘子拿下来洗，同时有人不断地把盘子摞在顶上。

静静发现每个盘子的价值是不一样的，于是他就很好奇，在他还没洗的盘子里和在他洗过的盘子里，最贵的盘子是值多少钱的？

Input

测试数据有多组，数据第一行包含测试数据的数目 T(≤10) 。

每组数据一开始输入一个正整数 n(n≤1000000) 。 
其下有 n 行，每行以1或者2开头，表示 n 个按照时间顺序发生的事件。 
如果以1开头，其后会有一个整数 v(1<v<1000000000) ，表示有人把一个价值为 v 的盘子摞在要洗的盘子顶上； 
如果以2开头，表示张静静把最顶的盘子拿去洗了。

起初，张静静没有任何盘子要洗。数据保证合法，即不会在静静没有盘子要洗的时候出现2。

Output

每组数据输出n行，表示对每个时间的响应。 
对于每个1输出现在还没洗的盘子（包括现在刚放入的盘子）中最贵盘子的价值。 
对于每个2输出当前已经洗了的盘子（包括刚刚拿走的盘子）中最贵盘子的价值。

Sample Input

1
5
1 1
1 3
2
1 2
2

Sample Output

1
3
3
2
3
