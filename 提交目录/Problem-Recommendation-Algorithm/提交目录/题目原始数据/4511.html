Description

I have a number of piles of bananas, and you’d like to play a game with me and get these bananas. But I want to get it harder so I decide the rules as follows:

 

1) For each pile i, we have Pi bananas.

2) Each turn we two can only take at most m bananas at least one from one pile.

3) Whoever can not take any bananas is the loser.

4) I will let you go first.

 

Can you still be the winner?

Input

Input has multiple cases.

 

The first line of each case contains two positive integer n and m, where n indicates  the number of piles of bananas and m is the max number of bananas you take at a time.

The following line contains n positive integers P1..Pn, Pi is the number of bananas in 

pile i.

   Every input is from 1 to 1000000.

Output

If you can be the winner, output “Win”

If you will be the loser, output “Lose”

   Note: I am a perfect player.

Sample Input

1 5<br/>6<br/>2 3<br/>1 5

Sample Output

Lose<br/>Lose
