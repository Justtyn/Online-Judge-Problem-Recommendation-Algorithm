Description

小Q最近在健身，今天他又来到了健身房准备健身；他只做三种类型的健身练习，分别是胸肌练习，手臂练习，背部练习；另外，小Q的训练是循环的，他第一个做的练习会是胸肌练习，第二个是手臂练习，第三个背部练习，第四个是胸肌练习，第五个是手臂练习，以此类推，直到他做完n个练习；现在小Q需要你帮忙了，给出小Q做的n个练习各做了多少分钟，小Q希望你帮他求出哪种类型的练习做了最多时间？

Input

输入有多组数据，每组数据首先是一个整数n（1<=n<=20），代表有n个练习，接下来一行有n个整数a1,a2...an(1<=ai<=25),代表第i个练习所做了多少分钟；当n=0时，表示测试结束；

Output

每组测试数据输出一行，如果胸肌练习做了最多时间，输出"chest"，如果是手臂练习做了最多时间，输出"biceps",如果是背部练习做了最多时间，输出"back";（保证是唯一解）

Sample Input

2
2 8
3
5 1 10
7
3 3 2 7 9 6 8
0

Sample Output

biceps
back
chest
