Description

    话说,话说,Miaowu是一只十足的吃货,尤其是吃各种各样的果果.某一天,很多很多Miaowu得到了很多很多的果果,排成一排,好长好长的一排,可现在很多很多的Miaowu犹豫了,到底应该吃哪些呢??!!!!所以每只Maiowu决定随便选一个区间,吃掉里面价值不同的果果,因为这么好吃的果果,如果不尽可能的多尝一些口味,这是何等的遗憾呀,比如排成一排的果果的价值是:

1 1 2 2 3 3 4 4 那么在区间[1,8]内,喵呜可以吃到的果果的价值是1+2+3+4=10的总价值.

好了,现在有n个果果,排成一排,第i个果果的价值是a[i],假设,某一个果果被吃掉后,该点会立马产生一个相同的果果.然后有m只Miaowu,并且已经知道了每只Miaowu会选择哪个区间,,所以请教你能不能事先得出每只Miaowu能得到的价值.

Input

多组数据(数据数<=60),对于每组数据第一行包含2个整数n,m(1<= n,m <=100000),然后有一行包含n个整数,a[i]表示第i个果果的价值,(0 <= a[i] <=10000),接着又m行,每行2个数字x,y,表示这只Miaowu会选择闭区间[x,y]的果果来吃. (1<= x<=y <=n)

数据量大，建议使用scanf。

Output

对于每组数据输出m行,每一行输出一个整数,代表这只Miaowu可以吃到的果果价值.
