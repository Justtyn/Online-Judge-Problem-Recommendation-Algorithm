Description

“编码字典”是一种常见的数据压缩方法，用字典的序数值来表示重复单词在文本中的的位置。
其实静态字典编码也可以解决这个问题，但是搜索的字典必须预先已知的，需要前期准备一个字典。动态字典编码避免了这个问题，从开始到结束，首先将第一个单词插入到一个空字典中。每当遇到重复出现的单词，它被替换为字典中的序数值。每当遇到单词未在字典中出现，则被添加到字典的末尾。

Input

若干行，第一行一个整数n，表示共有n个段落文本需要数据压缩。
以下每个段落有若干行，文本只包含小写字母和空格。任意单词长度不超过20，每行不超过80个字符，所有输入不超过100行。段落之间用一个空行分隔，且互相之间是独立地被压缩。

Output

若干行，对应使用动态字典压缩的输入文本的段落，空行和空格应予以保留。

Sample Input

1
the cat chased the rat while
the dog chased the cat into the rat house

Sample Output

the cat chased 1 rat while
1 dog 3 1 2  into 1 4 house
