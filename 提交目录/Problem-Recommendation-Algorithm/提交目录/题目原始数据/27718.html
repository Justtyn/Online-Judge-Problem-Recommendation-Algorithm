Description

键盘输入一个高精度的正整数N，去掉其中任意M个数字后剩下的数字按原左右次序将组成一个新的正整数。编程对给定的N和M寻找一种方案使得剩下的数字组成的新数最小。输出组成的新的正整数。

输入数据均不需判错。如果去掉了某几个位后得到的新整数开头为0，保留0。

Input

第一行为高精度正整数N（N的长度不超过10^6位）

第二行为M（0<=M<=N的长度）

Output

去掉M位后的最小新数。

Sample Input

【样例输入1】
82386782
3

【样例输入2】
505
1

Sample Output

【样例输出1】
23672

【样例输出】
05
