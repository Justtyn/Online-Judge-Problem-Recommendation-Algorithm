现在需要里做一个简单的查询程序，用于查询屏幕分辨率所对应的英文缩写。

Input

 输入有若干行，每行有两个数字w h(0 < w,h < 10000)，用空格分开，表示屏幕分辨率。每行的输入不保证都是上表存在的屏幕分辨率，输入的宽和高可能是对调的，即640 480与480 640应该表示相同的分辨率类型。输入为0 0时结束。

Output

 对应每行输入分辨率，输出一行屏幕类型英文缩写，如果所输入的分辨率在上表中不存在，输出“*Unknown*”（不包括引号）。
