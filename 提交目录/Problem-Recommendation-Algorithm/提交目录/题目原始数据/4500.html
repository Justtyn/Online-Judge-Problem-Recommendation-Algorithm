Description

众所周知，如今山寨货正大行其道。其中，很多山寨货都在品牌名称和包装上做到与原品牌极为相似，几乎能以假乱真，令购买者难以分辨。像“脉动vs脉劫”，“王老吉vs壬者吉”，“雪碧vs雲碧”.......

TiaoTiao同学由于视力不好，他幼小的心灵就曾多次被山寨货所欺骗。为了避免再次买到山寨货，他希望有人帮他做一个调查 —— 从货物的名称上，统计一下如今各大正品品牌被山寨的程度。

判断某品牌名是否山寨了一个原品牌名，可依靠以下规则 :

（设Origin为原品牌名，Unknown为待判定品牌名）

1.  若Unknown相比Origin多一字符，并且去除Unknown中的一个字符后，Unknown和Origin完全一样，则可认为Unknown为Origin的山寨名称。
|( 如，“nikke”是“nike”的山寨名称 )

2.  若Unknown相比Origin少一字符，并且Unknown增加一个字符后，Unknown和Origin完全一样，则可认为Unknown为Origin的山寨名称。
|( 如，“chanel”是“channel”的山寨名称 )

3.  若Unknown和Origin长度一样（字符个数相同），且Unknown相对于Origin只拼错了一个字符并且这个字母所在两个字符串的位置相同，则可认为Unknown为Origin的山寨名称。
( 如，“omaga”是“omega”的山寨名称 )

4.  除上述三点外其他的关系，Unknown均不被视作Origin的山寨名称。

（如果两个字符串完全相同的情况不叫山寨）

现在，给出一组原品牌名和一组待判定的品牌名，请统计出各原品牌被山寨的程度 —— 即分别统计出各原牌名称总共被山寨的次数。

(名称忽略大小写，例如，“abc”和“aBC”是完全一样的名称。同时名称只由字母组成，不包括空格或者其余符号。 )

Input

输入第一行，输入一个整数N，表示共有N个原品牌名，接着输入N行，每行输入一原品牌名。（N = 0的时候结束）

接下来一行，输入一个整数M，表示共有M个待判定品牌名，接着输入M行，每行输入一判定品牌名。

N,M的范围在[1,100].每个字符串的范围不超过20，

注 : 输入保证任一待判定品牌名最多只山寨了一个原品牌名，即保证不会出现有如以下的输入情况.

2

Sun

Sony

1

Son

Output

按照原品牌名的输入的顺序，一次输出原拼牌被山寨的次数。

输出“Name=X”，其中，Name为原品牌名（大小写应和输入时保持一致），X为一个整数 —— 表示此原品牌被山寨的次数。

Sample Input

3<br/>MaiDong<br/>WPS<br/>BS<br/>6<br/>MaiDog<br/>WCG<br/>BBS<br/>MaiDou<br/>CS<br/>BS<br/>0<br/>

Sample Output

MaiDong=1<br/>WPS=0<br/>BS=2
