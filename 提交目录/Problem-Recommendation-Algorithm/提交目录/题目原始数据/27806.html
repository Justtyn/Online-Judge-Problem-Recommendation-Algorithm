27806: B

Submit: 3  Solved: 2

Description

定义幸运数字为只含有4和7的数字，例如4，7，47，777等是幸运数字， 5，72等就不是幸运数字

定义函数next(n)表示最小的大于等于n的幸运数字，现在给你一个a,b让你求next(a)+next(a+1)+....next(b-1)+next(b)的值。

Input

第一行输入一个数T，表示测试样例个数，对于每个测试样例，输入两个数a，b(1<=a<=b<=1e9)

Output

对于每个测试样例，输出一个数，表示next(a)+next(a+1)+...+next(b-1)+next(b)的值，每个测试数据占一行

Sample Input

2
2 7
6 7

Sample Output

33
14

HINT


第一个测试样例:next(2)+next(3)+next(4)+next(5)+next(6)+next(7)=4+4+4+7+7+7=33
