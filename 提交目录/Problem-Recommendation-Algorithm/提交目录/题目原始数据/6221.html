Description

小D对身边事物都比较留意，而且时常会联想到一些问题的答案， 并会问号称“百事通”的小Q怎么算， 而每次小Q都能迎刃而解。

有一天， 小D与小Q出去逛街， 看到一些奇怪的墙。 每块墙都是直接由一块块砖叠起来的，并且参差不齐。

小D想着， 如果可以将这块墙一行行切下来， 会变成多少块， 然后不由自主的问起了小Q。小Q算10列以下的墙还可以算出结果， 但是要算几十列的墙就崩溃了。小Q知道，这只能编程实现， 人脑是跟不上的。 聪明的小Q知道BNUZ今天举办ACM新生赛，于是想请教聪明的你。 快帮帮小Q吧！

如下图1为3，5，4，2，4的五列墙。

表示的是第一列有3块砖，第二列有5块砖，第三列有4块砖，第四列有2块砖，第五列有4块砖。（注意：这些砖是完全一样的）

将这块墙一行行切下后，会变成7块。如图2.

 

Input

有T组数据。(T <= 15)。

每组数据的第一行，有一个n（n <= 100,000).， 表示一块N列的墙。

接着一行，有n个正整数ai, 第i个整数表示的是第i列由多少块砖叠成（1<= ai <= 100,000)。

Output

对于每组数据， 输出一个整数，表示会变成多少块。

Sample Input

2
5
3 5 4 2 4
5
5 5 5 5 5

Sample Output

7
5
