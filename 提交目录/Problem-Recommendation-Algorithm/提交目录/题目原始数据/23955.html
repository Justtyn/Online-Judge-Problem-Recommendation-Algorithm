Description

随着信息时代的到来，学生们的交流主要在网络平台上，例如比较流行的Facebook. 在Facebook中有个比较有趣的问题是：两个人之间的“分离度”是多少.
例如在下面的学生关系表中，Abby 和 Alberto 互不相识，但是可以通过下面的途径交流信息:
• Abby → Zoey → Alberto
• Abby → Natalie → Zoey → Alberto
• Abby → George → Ali → Kara → Richardo → Jeff → Alberto
通过观察我们发现，Abby 和 Alberto之间信息交流的最短路径长度为2（Abby → Zoey → Alberto）,因此我们就说分离度是2.换句话说，Alberto 是 Abby 朋友的朋友.

首先确定本题一直延续上表中的人际关系，请根据输入的相关操作，进行处理，进一步理清其中的关系。

Input

输入的数据包含6中操作，分为操作符和参数。首先确定X,Y (X<>Y,X>=1，Y>=1，X<50且Y<50)
• i x y – 建立一种人际关系，使X和Y为朋友。X和Y可能为新人。
• d x y – 解散X和Y之间的朋友关系.
• n x – 输出X的朋友人数.
• f x – 输出X的朋友的朋友人数（即间接朋友）.
• s x y – 输出X和Y之间的分离度，如果为0，则输出‘Not connected’.
• q – 退出程序.
