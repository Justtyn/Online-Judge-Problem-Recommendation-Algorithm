Description

每当Jo Coder 感到无聊的时候，她总是喜欢在桌上玩硬币游戏。她首先取出一组N（N<5）个硬币排成一行。然后按照以下规则进行游戏：
• 初始每个硬币有一个独立的位置，游戏中途不得新增加位置。游戏过程中可能出现某些位置没有硬币为空的情况。
• Jo Coder 可以把硬币移动到相邻的位置。
• 这些硬币可以堆叠，Jo Coder 可以从一个堆的顶部取一个硬币移动到另一个堆的顶部。
•堆叠的硬币严格按照从小到大的顺序，大的在下，小的在上。
