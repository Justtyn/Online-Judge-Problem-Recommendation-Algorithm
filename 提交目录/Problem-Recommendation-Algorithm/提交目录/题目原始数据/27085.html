27085: 学习时间

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 4  Solved: 1

Description

晨晨学校要统计学生在周末的学习情况，让每位学生自己写下在家里面的学习时间
段，带回学校。已知学校总共有 N (1 <= N <= 145)位学生参与调查，学生按照下面的格
式来记录学习时间：整数 c（代表学生学号） 、一个关键字"start"或者“stop” （代表此信
息是开始还是结束） 、两个整数 hh 和 mm（代表开始的小时和分钟，0 <= HH <= 23; 0 <= MM
<= 59） 。每个学生只是记录一天的学习时间，而且保证信息都是完整的，也就是说一位学
生有开始时间必然有相应的结束时间对应。
你的任务就是帮助计算每一位学生一天花费在学习上的时间是多少。
举一个例子：有下面信息：
2 6
1 START 9 0
2 START 9 30
1 STOP 10 0
2 STOP 10 15
1 START 17 0
1 STOP 17 42
其中第一行代表有 2 个学生提供了数据， 总共有 6 段时间在学习。 学生 1 在 9:00-10:00
和 17:00-17:42 时间段在学习，总共花费了一小时 42 分在学习；学生 2 在 9:30-10:15 时
间段学习，总共花费了 45 分钟。

输入格式 ：
第一行：两个整数 n 和 Nlines（代表信息条数 1 <= Nlines <= 1,458）
第 2..Nlines+1 行：每一行为一条学生学习信息。
输出格式 ：
总共 n 行，每一行两个整数代表第 i 位花在学习的小时和分钟。

2 6
1 START 9 0
2 START 9 30
1 STOP 10 0
2 STOP 10 15
1 START 17 0
1 STOP 17 42

Output

1 42
0 45
