介绍完后，小丽忽然说了一句,三十九阶台阶，你有多少种方法上去，小磊深知，自己腿短，自己一下子最多能上两节，小磊先算上一阶台阶有一种方法能上去，上两阶有两种。。。。小磊算了一个晚上也没有算到第39阶，于是就去问被人称作学霸的ACM的同学，为了不失面子，当小磊问你上 n 阶台阶有多少种方法时，你能快速告诉他么？

Input

第一行是一个整数 t ，表示有 t 组测试数据；
每组测试数据只有一个整数 n，表示有 n 阶台阶（0 < n < 40）;

Output

对于每组测试数据 n ，输出一个整数，表示有多少种方法可以上去；

Sample Input

2
1
2

Sample Output

1
2

HINT

当 n = 2 时，小磊可以一步跨两个台阶；也可以先跨一个台阶，再跨一个台阶。所以输出结果为 2
