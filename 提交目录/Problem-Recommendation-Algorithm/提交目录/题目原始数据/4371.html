Description

小佳姐姐有一串珠子由红，黑，白三种不同颜色组成，R表示红色珠子，B表示黑色珠子，W表示白色珠子，但由于莫名其妙的原因，这串珠子断成两截（每节至少有一颗珠子）。

现在，佳姐想将珠子复原为闭合的珠链，但由于佳姐有特殊的爱好，她喜欢珠子以红黑相间出现，并且认为白色珠子可看成任意颜色（即红或黑），现在，给定两个由字母R,B,W组成的珠子串，问，复原成一个闭合的珠串后，这种红黑相间的珠子（包括可看成任意颜色的白色珠子）最多有几个。

特别提示：白色珠子可认为是红黑珠子中任意一种，包含进后面的计数。

Input

输入第一行有一个正整数T，表示有T组测试数据。

接下来每组测试数据有两行，每行有一个由R,B,W组成的字符串（最多100个字符），表示断截的珠子。

Output

每组测试数据输出一行。

如果不能组成任何红黑相间的形式，则输出NO。

如果复原后的闭合珠链全部由红黑相间则输出YES。

其他情况，则输出红黑相间珠子的最多个数。

Sample Input

6
RWWWRBBB
BBRW
BRWRWBBBRB
WB
RR
RBB
B
BRRWBWRBBR
BBBWWBBR
WBWBBBW
RWR
WB

9
7
2
3
5
YES
