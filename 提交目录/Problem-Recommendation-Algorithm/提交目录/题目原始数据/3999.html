Description

最近喵呜纠结数学，结果走火入魔，不能自拔。各位速度去营救吧！

喵呜在一个N*N的陷入一个矩阵中。矩阵里充满了数字，此时这些数字化身成了可怕的恶魔（由于喵呜走火入魔所生，悲剧呀。。。）矩阵的左上角是唯一的入口，在入口你会得到喵呜不慎留下的《高等数学》，提升你的战斗力。战斗力为（0，0）处的数值（数值大于等于0）。矩阵其他格子中有数字化生的恶魔，战斗力为其中的数值（不超过int32，且大于等于0）。你必须打败它们才能前进。数字世界中的战斗方式有所不同。你的战斗力与恶魔的战斗力的二进制表示中，不同的数位的个数记为k（如101  vs  110，k=2），当k大于已给定值a时，你能够战胜恶魔，否则战败。

只可直走，不能斜行。那么请找到一条最短路线，营救喵呜！！行动吧！！

Input

第一行 N<500   a<32

第二行~第N+1行每行N个数（喵呜的位置用-1表示）

Output

如果营救成功，以坐标形式输出最短路线。如果有多条路线，输出字典序最小的一条路径。详见样例。

如果营救失败，输出“miao”

Sample Input

4 1
3352 22636 23512 14032
30178 2473 22667 5551
18327 472 -1 28023
27876 22590 17533 29514

Sample Output

(0,0)(0,1)(0,2)(1,2)(2,2)
