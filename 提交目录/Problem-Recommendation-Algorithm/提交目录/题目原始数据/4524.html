Description

江湖上有位豪杰名曰令狐冲。令狐大侠行侠仗义，豪放不羁，是多位女神的梦中情人。华山派的岳灵珊，与令狐大侠青梅竹马；恒山派的冰清玉洁的仪琳，对令狐大侠相思痴恋；魔教的东方菇凉，与大侠多次互相解救而一往情深；魔教的圣姑，娇美慧颉的任盈盈姑娘，则能与令狐大侠一同演奏笑傲江湖之曲，堪称知己。

   世间的感情讲求缘分二字，有一天4位女神和令狐大侠齐聚到一起，做了一个缘分测试游戏。他们每个人手上都有一个卡片，每个人写一个数字num (1<= num <=1000)。女神中卡片上的数字与令狐冲的卡片上的数字最接近的人，是与令狐大侠最有缘分的人，最终也赢得了令狐大侠的心。

Input

第一行，一个数字T（T<=1000），表示有T组测试数据。

对于每一组测试数据，输入5个数字，分别代表令狐冲，岳灵珊，仪琳,东方菇凉，盈盈的卡片上的数字。

Output

对于每一组测试数据，如果灵珊师妹跟令狐大侠更有缘，则输出“Yue Lingshan win!”,如果仪琳赢了， 则输出“Yi Lin win!”，如果东方菇凉赢了，则输出“Eastgod will never lose!”，如果盈盈赢了，则输出“Ying Ying win!”。如果有多位女神与令狐大侠最有缘，则输出“Ling Hu is crazy!”

Sample Input

3
1 2 3 4 5
500 200 400 499 800
30 2 8 20 40

Sample Output

Yue Lingshan win!
Eastgod will never lose!
Ling Hu is crazy!
