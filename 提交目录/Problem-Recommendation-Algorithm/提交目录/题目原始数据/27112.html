Description

 假设有一列数 {Ai }(1 ≤ i ≤ n)，n<=100000 ，支持如下两种操作：

(1)将 A i至A j 的值均增加 D 。（ i,j,D 是输入的数）

(2) 输出 Ai 。（ i是输入的数， i ≤ n ）

根据操作要求进行正确操作并输出结果。

Input

 输入文件第一行一个整数 n ，

第二行为 n 个整数，表示 {A i } 的初始值。

第三行为一个整数 m ，表示操作数。 下接 m 行，每行描述一个操作，有如下两种情况：

ADD i j d ( 表示将 A i至A j 的值均增加 D ， 1<=i,j<=n ， d 为整数 )

QUERY s （表示输出 A s）

Output

 对于每一个 QUERY 提问，输出结果

4

3

QUERY 2
