Description

无聊的时候小东东喜欢摇一摇，相信大家也都玩过摇一摇。咔咔两下，吼吼~出现了好多GG还有MM。不过小东东肯定最喜欢MM啦。而且小东东只想和距离他一定范围内的MM聊天。所以呢，现在请你计算一下小东东每次能挑选MM的个数。

 

Input

   第一行一个T表示小东东摇了T次T<=20。紧接着的下一行包含四个整数N X Y M，N表示当前在玩摇一摇的人数1<=N<=10000。（X，Y）表示当前小东东所在的坐标。M是他想过滤的范围10<=M<=1000，当然他希望摇出来的人距离自己都在M之内。接下来的N行每行三个参数X Y S。（X，Y）表示正在摇一摇的人的坐标，S标识此人是MM还是GG。(0<=X,Y<=10000)

Output

请计算出小东东每次摇一摇可选MM的个数。

Sample Input

3
3 0 0 10
10 0 MM
5 5 MM
8 6 GG
1 100 100 1000
1000 1000 MM
4 5 7 10
3 5 MM
3 5 GG
0 0 MM
57 28 MM

Sample Output

Case #1: 1
Case #2: 0
Case #3: 2
