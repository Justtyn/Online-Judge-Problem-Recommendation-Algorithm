Problem 4591 --G、大丈夫大厦的电梯

4591: G、大丈夫大厦的电梯

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 0  Solved: 0

Description

 有一系列大厦取名叫大丈夫，只因为他们能屈能伸。大厦有若干层，每一层高度可以随意自由变化，低层的高度被改变时，高层可以安全地上升下降。但是这给你所在的电梯制造商提出了麻烦的问题——怎么正确显示电梯某一时刻在第几层。你公司制造的电梯一直以来通过电梯所在高度来计算所在楼层，因为目前只能得到这个的实时数据，只能在每层层高相同的建筑上使用。DS攻城师提出的方案是每个楼层加装感应器，再通过一系列复杂的软件开发来解决这一问题，这将大大增加成本投入。但你发现这座大厦的智能控制系统软件可以在楼层高度发生改变时发出通知，只需要很少的软件开发量就能实现在原有的硬件基础上得到某时刻电梯所在楼层。

大丈夫系列大厦在各地所建楼层层数不同，都很多层，而且楼层高度发生变化的情况经常发生，很多层的电梯显示器也频繁向你所开发的监控软件获取电梯所在楼层信息，你所开发的软件必须反应快速结果精确。最开始时，每层楼的高度都是5，大厦的楼层从1开始算起，楼层高度从第一层地板算起，电梯楼层以电梯地板到达某一楼层地板相同高度或大于某一楼层地板高度小于更高一楼层地板高度为准，如第一层高度为5，则电梯地板位于0到4之间时，认为电梯处于1楼。

Input

 有若干组测试数据，每组数据的第一行有两个正整数N(0<N<=50000)和M(0<M<=1000000)，表示某个大丈夫大厦有N层，然后接下来有M行。接下来的M行有询问楼层和楼层高度变更两种操作，每一行都由非负整数A和B组成。当A为0时，你需要输出位于B(0<B<当前时刻所有楼层高度的加和)高度的电梯，目前在第几楼；当A>0时，表示楼层A的高度变成B(0<B<=100)。

当输入的N和M同时为0时，结束程序。

Output

 当输入的操作为询问楼层时，输出一行，只有一个数字，表示询问所得到的楼层。每组测试数据结束，输出一个空行。

Sample Input

4 5
0 5
1 6
0 5
0 11
0 10
2 1
0 7
0 0

Sample Output

2
1
3
2
2
