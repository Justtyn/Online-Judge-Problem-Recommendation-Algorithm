Description

在一个二维平面上有两类点，一类称为A类点，另一类称为B类点，其中A类点有n个，B类点有m个；现在想在以这些B类点为圆心画圆，且每个圆的半径r一样，如果希望所有A类点至少被一个圆覆盖（点在圆内部或者点在圆上），那么r的最小值是多少，精确到小数点后面四位。

Input

输入数据有多组，直至文件末尾。
每组输入数据的第一行有两个整数n和m，分别表示A类点的个数，和B类点的个数。
接下来有n行，每行两个整数表示一个A类点的坐标，输入数据保证A类点在一条直线上。
接下来有m行，每行两个整数表示一个B类点的坐标，输入数据保证B类点在一条直线上。
数据范围：1<= n<= 100000, 1<= m<= 100000, x坐标和y坐标均在32位整数范围以内。
输入数据不保证没有重点，也不保证A类点的直线和B类点的直线不相同。

Output

输出：对于每组数据输出一行，包含一个实数，表示满足条件的最小半径，请一定要记得保留四位小数。

Sample Input

2 1
1 0
-1 0
0 1

2 2
1 0
-1 0
1 0
-1 0

Sample Output

1.4142
0.0000
