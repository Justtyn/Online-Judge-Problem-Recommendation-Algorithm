Description

　　学校里的楼房真是盖得奇形怪状，而楼房里水管的布局也成了难题。水管工人需要常常修理大楼的水管，而许多大楼里的水管是错综复杂的，使得水管工人很难弄清水管的走向。 　　大楼里的水管总是从上往下的。

如图所示是一个4层楼4条水管的大楼。顶楼和底楼总是对应着水管的两端，并排成一行。而在中间的楼层里，水管可能相互交叉。幸运的是，每层楼每条水管最多交错一次，并且只能和相邻的水管相交。 　　水管工人在大楼顶部对水管从1到n依次编了号，然后一层一层往下走。由于太复杂，水管工人记不住当前楼层的水管到底是什么编号，他只知道这层楼哪个位置的水管和哪个位置的水管相交。 　　每到一层，水管工人就会记录下这层楼所有水管的交换情况。例如，第一层时，1和2位置的水管相交，3和4位置水管不变，记为2 1 3 4；第二层时，1位置和4位置水管不变，2和3位置水管相交，记为1 3 2 4。 　　当水管工人下到底楼时，发现他所记录的数据实在太复杂了，他希望你能帮助他确定底楼水管所对应的编号。

Input

　　第一行包含两个正整数n m (1<=n,m<=10)，表示总共n条水管，m层楼。接下来的m行，每行包含n个数，描述了该层楼水管的布局情况。输入数据保证符合题目描述。

Output

　　输出一行，包含n个数，表示底楼水管所对应的编号，每个数后输出一个空格。输出完毕请换行。

Sample Input

4 4<br/>2 1 3 4<br/>1 3 2 4<br/>2 1 4 3<br/>1 2 4 3<br/><br/>

Sample Output

3 2 1 4<br/>
