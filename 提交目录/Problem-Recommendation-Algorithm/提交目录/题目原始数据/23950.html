Description

加拿大的顶级核科学家，Patrick 和 Roland，刚刚建成了世界上第一个核裂变反应堆。现在，他们的工作就是每天24小时不间断地操作和测试反应器。这样的工作持续了一段时间后，两位科学家都感到有点厌倦，而且这期间发生了两件事情。首先，他们现在可以控制反应器内发生的单个反应。其次，为了打发时间，他们发明了一种新的称为Nukit的游戏。
Nukit游戏开始前，首先将一些粒子放入反应器中。两位科学家轮流操作，Patrick总是先走。轮到一位科学家走的时候，他必须从可以发生反应方案的中选取一个进行操作。然后，这些粒子被消耗掉，随着反应的持续粒子将越来越来。如果轮到哪一位操作时无法进行任意一种反应，则判定他这一局为输。
在我们的宇宙中假定只有四种粒子，分别标注为: A, B, C, D. 目前反应器中可以进行的反应实验有以下五种:
1. AABDD
2. ABCD
3. CCD
4. BBB
5. AD

例如第一种方案，我们可以选择2个A,1个B,2个D粒子进行反应操作，操作完成后这些粒子将被消耗掉。
事实证明，不管有多少个粒子，游戏最终会决出胜负，Patrick 和 Roland 其中的一人将赢得游戏。例如，当有1个A,5个B,3个D粒子，不管Patrick 如果操作，Roland将最终获胜。
给出每一局游戏开始时每种粒子的个数，请编程计算每一局谁将最终获胜。

Input

第一行，一个整数n(1 ≤ n < 100)，表示共有n局游戏。
接下来n行，每行4个整数，互相之间用空格隔开，分别表示A,B,C,D的粒子数，范围为0..8。

Output

共n行，每行一个字符串，表示每一局获胜者的名字 “Roland” 或 “Patrick”.

Sample Input

6
0 2 0 2
1 3 1 3
1 5 0 3
3 3 3 3
8 8 6 7
8 8 8 8

Sample Output

Roland
Patrick
Roland
Roland
Roland
Patrick
