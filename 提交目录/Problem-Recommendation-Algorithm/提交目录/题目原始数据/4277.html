4277: 冰上城市

Description

　　探险者CGC来到一座被冰雪覆盖的城市，CGC希望在这个城市里买到一些补给物以便能继续他的探险旅程。 　　城市里有些地面结上了厚厚的冰层。冰面是很滑的，城市里的人们不愿意总是在冰面上溜来溜去，于是他们修建了一栋很大的楼房，使得所有的居民都生活在里面。 　　CGC试着向前方冰面走了一步，由于冰面太滑，CGC没办法使自己停下来，一直向前滑行，直到撞到了对面的墙壁才停下来。 　　这下CGC可犯难了，他想知道如何才能进入到这个城市的大楼。 　　CGC从好心的路人Tiaotiao的口中了解到，整个城市可以看作一个n×m的方阵，每个格子不是墙壁就是路面。墙壁是不能通过的。路面都是普通的平地，不会打滑。可是由于很久以前城市来了一头大象，这头大象喷出的冷气使得很多路面都冻结了！城市从此以后常年结冰，城里的居民出行都十分不便。 　　在冰面上行走时，由于太滑，无法停下来，只能沿着行走的方向一直滑行向前，直到滑出冰面或是撞到墙壁。若滑出冰面到了普通的路面，那么行人将停在普通路面上；若滑行时撞到墙壁，行人将会停留在墙壁旁边。 　　CGC希望你能帮助他，让他行走最少的步数进入到冰上城市的大楼。CGC只能向前后左右四个方向行走；在冰上行走时，从滑行到停下算作一步。

Input

　　输入包含多组测试数据。每组测试数据第一行包含两个整数n,m表示城市的大小(2<=n,m<=100)。接下来的n行每行包含m个字符，描述这个城市。字符’.’表示普通路面，’/’表示冰面，’#’表示墙壁。地图上唯一一个字符’H’表示大楼的入口，’C’表示CGC所在的位置。 　　认为在整个城市外有一圈墙壁包围，CGC所在的位置是普通路面。 　　输入以n=m=0结束。

Output

　　对于每组测试数据输出一个整数，表示CGC最少需要行走多少步才能进入大楼。若Carl无论如何也进入不了大楼，则输出’Impossible’。输出格式参照示例。

Sample Input

2 6
H....C
./////

4 4
#H#C
#///
.//.
#//.

3 3
//C
/H/
///

0 0

Sample Output

Case 1: 3
Case 2: 4
Case 3: Impossible
