但是，在旅途中，克劳斯的列车要经过各种各样的神奇地域，每个地区因为地形和地质的原因，使得列车的行进速度都不尽相同，为简单起见我们假定极地特快在任何地区总能大致走直线，而克劳斯的家在世界地图的左下角，极地旅途的终点在地图的右上角（如下图所示），最后我们也知道了这趟旅途所经过的所有地区构成了一个w*h的矩形，每一个地区我们将给出它们的水平跨度wi，和在这个地区旅行的速度vi。我们知道克劳斯和它的伙伴们难以按奈一睹极地风光的渴望，因此他们想找到一条列车行进的路径，使他们能够以最快的时间到达极地！你能帮助他们吗？

Input

对于每组测试数据：

首先给出3个整数 n,w,h  (1<= n <= 100， 1<= w,h <= 60000)
