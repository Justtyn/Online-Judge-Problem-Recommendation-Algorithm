Description

Given a n*m matrix consist of integers, you can do some operates on this matrix at most 10086 times. Each time you can select one row or one column and change all the element in this row (column) to its opposite number. Your task is to find whether you can change this matrix into a new one that the value of each element in this matrix is non-negative.

Please note that the opposite number of x is –x.

Input

The first line contains only one integer T, which is the number of test cases.
Each test case starts with two integers’ n and m which describe the size of the given matrix.

Then follow n lines, each line consists of m integers described the element of this matrix.

 

Output

For each test case, output the case number first, if you can change this matrix into a new one that the value of each element in this matrix is non-negative please output “Yes”, otherwise please output “No”.

 

Sample Input

3
2 2
1 -1
-1 1
2 2
1 1
1 -1
2 2
0 0
0 0

Sample Output

Case 1: Yes
Case 2: No
Case 3: Yes

HINT


T<=100



n, m<=100



The value of each element of the matrix <= 100
