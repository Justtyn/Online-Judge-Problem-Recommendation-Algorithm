Description

有一天，小Y被派去采购K斤苹果。他需要空车从出发地出发，途径苹果供应商采购，并把苹果送到距离出发地E公里的目的地。

为了使题目简单，我们假设小Y的旅程是长度为E的坐标轴，出发地的坐标是0，目的地的坐标是E。小Y只能往出发地的方向行驶，途中停下来采购苹果，不能往反方向行驶。如果小Y载着p斤苹果行驶m公里，他需花费p*p*m元的运输成本。

出发地和目的地之间有N个苹果供应商，每个供应商都位于坐标0到坐标E之间。第i个苹果供应商有Fi斤苹果的库存，每斤苹果卖Ci元。特别要注意，苹果供应商可能位于同一位置。

小Y能停在任何一个苹果供应商前采购，并且采购任意的数量（当然必须在该苹果供应商的库存范围之内），采购的数量必须为整数。

小Y需要知道，他最少需要花费多少钱，才能采购到K斤苹果，并且运输到目的地。

Input

多组测试数据。测试数据的开头是一个正整数t，表示测试数据的组数。

每一组测试数据都以三个整数K、E、N开头，分别代表着采购的苹果的重量、目的地坐标和苹果供应商的个数。（1 <= K <= 10000, 1 <= E <= 500, 1 <= N <= 500）。接下来N行，第i行都有3个整数Xi、Fi和Ci,分别代表第i个苹果供应商的位置，库存和苹果单价。（0 < Xi < E，1 <= Fi <= 10000，1 <= Ci <= 10,000,000）。

Output

每组测试数据输出一行，该行有一个整数，代表小Y需要花费的最小值。

Sample Input

1
2 5 3
3 1 2
4 1 2
1 1 1

Sample Output

9

HINT


1、 小Y从出发地点出发，空车到坐标1。0花费。


2、 在第三个苹果供应商不买苹果，0花费


3、 空车到坐标3。0花费。


4、 在第一个苹果供应商买1斤苹果，花费2元。


5、 载着一斤苹果到坐标4，花费1元。


6、 在第二个苹果供应商买1斤苹果，花费2元。


7、 载着两斤苹果到终点坐标5，花费4元。


8、 以上是小Y花费最小的采购办法，最小花费为9元。
