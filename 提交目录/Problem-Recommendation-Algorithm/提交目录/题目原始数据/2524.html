Description

炼金术师家族为了复活至宝并创造“奇迹”，开启“零碧计划”。前克洛斯贝尔地区最强的游击士，“风之剑圣”亚里欧斯·马克莱因，为了有机会改变克洛斯贝尔令人绝望的现状，开始协助计划的执行。现在，他站在特别任务支援科一行四人面前，阻止他们继续调查。

支援科四人早些时候已经料到要和亚里欧斯兵刃相向，而且自知战胜亚里欧斯希望渺茫。但是他们曾见过亚里欧斯战斗，对其攻防习惯算得上了解。他们提前开了一个作战会议，研究对策。

讨论方便起见，用一个数据描述支援科的战斗状态：BP（Battle Point）。若BP归零，则不能继续战斗。此时，缇欧将使用技能“虚无领域”，短时（认为是0）屏蔽剑圣的伤害。艾莉趁机同时使用技能“极光之雨”，立即恢复一定BP，回复BP的多少与第几次使用有关。

亚里欧斯亦不是等闲之辈。他发起的每次攻击均带有“剑气”。每次攻击造成的伤害不同，设为DP。每次攻击支援科都不敢小觑。不妨设亚里欧斯发出一记伤害为d的攻击，此时支援科战斗状态为b：

若b >d， 支援科可以抵挡住这次攻击，但BP需要减少d。攻击的剑气会被罗伊德和兰迪驱散，不会造成伤害。

若b = d，支援科的BP降为0，艾莉和缇欧马上使用“虚无领域”和“极光之雨”回复BP，攻击的剑气会被罗伊德和兰迪驱散，不会造成伤害。

若b < d，支援科的BP降为0，艾莉和缇欧马上使用“虚无领域”和“极光之雨”回复BP，不妨设恢复到了c。剩余的d - b点伤害会转化为剑气继续攻击支援科。若d - b > c，重复上面的事情：支援科的BP降为0，艾莉和缇欧马上使用“虚无领域”和“极光之雨”回复BP。攻击剩余的d - b - c点伤害会转化为剑气继续攻击支援科。若d-b < c, 攻击的剑气会被罗伊德和兰迪驱散，不会造成伤害。若d - b = c, 支援科的BP降为0，艾莉和缇欧马上使用“虚无领域”和“极光之雨”回复BP，攻击的剑气会被罗伊德和兰迪驱散，不会造成伤害。

身为“剑圣”级别的人物，一场战斗中，亚里欧斯有M次机会使用技能“麒麟功”，在发出一次攻击时翻倍其伤害，仅对当前攻击有效。

现在，给出亚里欧斯的出招表，和艾莉每次“极光之雨”可以恢复的BP。为了准备万无一失，找到制胜的机会，队内的军师艾莉和天才程序员缇欧需要算出亚里欧斯最多会让支援科使用多少次“极光之雨”。你可以帮助她们吗？

ps：“极光之雨”和“虚无领域”只在BP = 0的时候被动使用。

Input

本问题有多组样例。每组样例的第一行是三个整数N，M，K。第二行是N个数，表示亚里欧斯会按顺序发出N次攻击，每次的DP数量。第三行是K个数，表示艾莉一共可以使用K次“极光之雨”，每次恢复的BP数量。其中：

0 < N, K < 1000, 0 <= M <= 100.

0 < N[i], K[i] < 5000

M = N = K = 0时输入结束。

Output

 每组样例输出一行，包含一个数，表示亚里欧斯最多会让支援科使用多少次“极光之雨”。

Sample Input

2 1 2
1 1
2 1
2 0 2
1 1
2 1
0 0 0

Sample Output

2
1
