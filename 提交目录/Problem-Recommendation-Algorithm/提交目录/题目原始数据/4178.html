Problem 4178 --教室占位问题

4178: 教室占位问题

Description

来大学之后，由于没有固定座位，大家都应该有过帮同学占位，或者同学帮你占位的经历。
假设教室里有N×M个座位，如图，每个座位有一个“舒适值”，这个“舒适值”为一个非负整数，当然，有一部分椅子可能是坏的，如果椅子是坏的，则无“舒适值”可言，该位置“舒适值”为-1。

 

由于每位同学对于位置的需求有些许差异：
 
1、 第1部分同学认为，教室前N/3（取整）行座位“舒适度”较高，若一个座位在前N/3行，行号为n，该座位的原有“舒适值”为e，则在这部分同学看来，该座位的“舒适值”为e+(N/3-n+1)。同时，他们比较鄙视后面N/3（取整）行座位，若一个座位在后N/3行，行号为n，该座位的原有“舒适值”为e，则在这部分同学看来，该座位的“舒适值”为e-(N/3-(N-n))。
2、 第2部分同学认为，教室后N/3（取整）行座位“舒适度”较高，若一个座位在后N/3行，行号为n，该座位的原有“舒适值”为e，则在这部分同学看来，该座位的“舒适值”为e+(N/3-(N-n))。同时，他们比较鄙视前面N/3（取整）行座位，若一个座位在前N/3行，行号为n，该座位的原有“舒适值”为e，则在这部分同学看来，该座位的“舒适值”为e-( N/3-n+1)。
3、 第3部分同学认为，教室哪个位子都差不多，所以不加任何个人需求因素，按实际教室座位的“舒适度”找位子。
 
现在，有一些同学来到该教室，每个同学表示为m:s，q，e。m:s表示该同学来到教室的时间，q表示该同学需要占位的个数(1<=m<=100)，其中，第一个位是为自己占的，q-1个位是为同学占的。
占位过程如下，首先该同学会寻找有q个在同一行的座位（“舒适值”小于0，则算断开，因为没有人愿意坐），并且该同学坐在找到的q个座位的最左边（即第一个），若寻找到多于1种符合要求，则该同学会寻找自己位置“舒适度”最高的q个连续座位占座。若没有找到连续的q个位置，则放弃帮同学占位，给自己寻找一个“舒适度”最高的座位。若所有座位都已被占或者剩下的座位“舒适度”均为负值，则选择离开教室。

Input

每组测试数据首先有一行N M K(1<=N,M<=50,0<=K<=100)，表示这教室有N行M列，后面有K个学生来占座。

接下来N行，每行有M个数，表示每个座位的“舒适度”。

最后K行，表示有K个同学来占座，每行为

“m:s q e”(0<=m,s<=59,1<=q<=100,1<=e<=3)，

表示该同学在m分钟，s秒来到教室，保证每个同学到教室时间不重合，需要占q个座位，e表示该同学为题目所描述的1，2，3中的一类。

当N,M,K都为0时，结束程序。

Output

请按题目输入的顺序输出他们找到座位的坐标，若找不到座位离开教室则输出“Ort”。

Sample Input

3 3 3
1 2 3
4 5 6
7 8 9
09:01 2 1
09:03 2 2
09:02 1 3
0 0 0

Sample Output

3 2
2 2
3 1
