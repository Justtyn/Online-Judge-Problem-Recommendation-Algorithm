Description

为了即将到来的MARS汽车大奖赛，彤彤正在精心检修她的赛车, 除此以外，她还想再买一些配件来提升赛车的性能。如果给她的赛车增添一个重量为M (1 <= M <= 1,000)的配件，则动力可以提升F(1 <= F <= 1,000,000)个单位。赛车配件商店现有N (1 <= N <= 20) 种配件，分别标上序号1..N。彤彤可以随意挑选其中的任意一种或多种，不过同种配件只能买1个。
第 P_i 个配件能为赛车增加动力 F_i (1 <= F_i <= 1,000,000) ，但是赛车的重量将M_i (1 <= M_i <= 1,000)。牛顿第二定律告诉我们F = MA, F为动力, M是重量, A是加速度. 如果彤彤想最大限度提升它的赛车的性能（且增加重量最小），她需要选择哪些配件。
赛车目前的配置是F=1500 ，M=100 。有四种配件可供选择，它们的基本配置如下：
          i   F_i    M_i
          1  250   25
          2  150    9
          3  120    5
          4  200    8
如果添加第2个配件，那么加速度则为
(1500+150)/(100+9)=1650/109=15.13761。

* 若干行，表示需要添加的配件编号，每行1个。注意请把这些编号按照从小到大的顺序输出。如果不需要添加配件，则输出'NONE' 。
