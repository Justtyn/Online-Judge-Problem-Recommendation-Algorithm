Description

    小罗去上海参加比赛，准备带一样礼物送给MM。现在就有上海特色的就是世博会纪念品了，小罗觉得有个变色杯子很有趣，主题是辉煌的浦东，杯子外表是一个一个的格子，当倒上热水的时候，含有特殊材料的格子就会变色，就可以看到杯子上好像有一座座大厦拔地而起，如果把水倒满就可以看到很壮观的影像了。

        但是，如果水没有倒满，只有热水漫过一半以上（包括一半）高度的格子才会变色，东方明珠就少了一截了。这个杯子上的方格每一列能变色的部分都是从最下面一格开始一直连续到某一个高度，到了这个高度上方就没有可以变色的方格了。

    已知杯子上有N列方格，和每列从下往上数有几个会变色的方格Ci，当前倒水的高度M毫米，和方格的边长A毫米，方格的间隙为B毫米，底部第一个方格与地面没有间隙，一个方格被漫过超过一半高度时就会变色，请求出总共有几个方格变色了。

Input

每组测试有两行输入

第一行分别是 N M A B

第二行有N个数，Ci

输入N为0时结束

所有数字都是整数

Output

对于每组测试输出一行

Sample Input

3 10 4 1<br/>1 2 3<br/>3 12 4 1<br/>1 2 3<br/>0<br/>

5<br/>6
