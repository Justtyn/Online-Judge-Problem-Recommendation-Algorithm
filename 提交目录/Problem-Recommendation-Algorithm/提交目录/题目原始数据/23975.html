Description

你想知道所喜欢的球队在一次小型锦标赛中夺冠的概率。共有4支球队参加比赛，整个锦标赛实行单循环制共举行6场比赛。每一场比赛结果可能是胜、负、平中的一种，胜得3分，平得1分，负得0分。
你喜欢的球队夺冠的条件是最终积分第1，比其他球队的积分都要高（并列时将不会被看作积分第一）。在锦标赛所有比赛结束之前没有人知道那支球队最终夺冠。请你根据已经举行的比赛情况，计算你所喜欢的球队有多少种可能夺冠的机会。

Input

第1行为你所喜欢的球队编号T(1<=T<=4)
第2行为已举行的比赛场数G(0<=G<=5)
第3..G+2行每行四个正整数A,B,SA,SB，互相之间用空格隔开，A,B表示两支球队的编号，SA,SB表示A,B两支球队比赛最终得分，得分多的为赢。

Output

共一行，一个整数表示你所喜欢的球队有多少种可能夺冠的机会。

Sample Input

3
3
1 3 7 5
3 4 0 8
2 4 2 2

Sample Output

0
