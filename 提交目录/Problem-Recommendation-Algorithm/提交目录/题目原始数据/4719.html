Description

有一天，Kaiser说了一个笑话：

教科书中说，水的元素构成是氢和氧，其化学分子式用H2O表示。实验证明这是错误的。以下是我的实验过程：   

找一结实的大铁桶，在里面放上半桶水。然后拿菜刀在水中快速切割（不得把水弄洒到铁桶外）。   

如果水是由氢和氧构成的，那么在刀的每次切割过程中就会有一部分水分子被切割成单独的氢分子和氧分子。   

大家知道，在我们日常的温度下氢和氧是气态的，所以被切割开的氢分子和氧分子就会挥发掉，水会逐渐减少。   

可是我快速切割了一小时，水却丝毫没有减少（计算时考虑了水的正常蒸发使水减少和我的汗水落入使水增多的因素）。这就证明水的元素构成绝不是是氢和氧！

 

Kaiser说为了庆祝圣诞，决定出一个很简单的问题：

现在我们看到，Kaiser已经把很多物质给切开了，并且坚信没有产生单质，我们知道在化学尺度下，基本粒子团的化合价与粒子团的电子云偏向有关，为简单起见我们规定粒子团的化合价即为团内基本元素的化合价代数和，例如H2O，可以认为其电荷性为0，即 (+1 * 2) + (-2 * 1) =0 ，现在我们要做的事情很简单，给定一个粒子团中各元素的数量，以及化合价，试求粒子团的电性如何。

Input

首先给出一个整数t，代表将有t组测试数据

 对于每一组测试数据，第一行我们给出粒子团中粒子的种类数n；接下来的n行中，每行将给出二个参数m，k 代表粒子团中显示k化合价的元素有m个。 参数间以一个空格隔开。

Output

对于每组测试数据，以一行输出一个整数，代表这个粒子团最终的化合价

Sample Input

2<br/>4<br/>4 1<br/>1 -3<br/>1 5<br/>3 -2<br/>2<br/>1 5<br/>4 -2<br/>

Sample Output

0<br/>-3<br/>
