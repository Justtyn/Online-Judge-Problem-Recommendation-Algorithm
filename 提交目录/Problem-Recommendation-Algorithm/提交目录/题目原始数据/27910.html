27910: 斜线迷宫

Description

    利用斜线(/)与反斜线(\)可用来产生一组迷宫，如下图所示：

        

    本题着重在探讨迷宫中的封闭路径，计算封闭路径总数及其中最长的路径长度，如上图中有两个封闭路径，路径长度是以小方格的数目来做计算。上图中的最长路径长度为16，最短的路径长度为4。

    （来源：UVa705）

Input

    输入资料会包含多组迷宫资讯，每组迷宫的第一行有两个整数w 与h (1 <= w, h <= 75)，表示迷宫的宽度与长度。接下来的h 行，每行有w 个'/'或'\'字元表示迷宫。当w = h = 0 表示测试资料结束。

Output

    请对每组测试资料输出一行"Maze #n:"，其中n表示迷宫编号，接下来输出"k Cycles; the longest has length l."，其中k表示封闭路径数，l表示最长封闭路径长度。若迷宫中没有任何封闭路径，请输出"There are no cycles."。 请在每组输出资料后多输出一空白行。

Sample Input

6 4
\//\\/
\///\/
//\\/\
\/\///
3 3
///
\//
\\\
0 0

Sample Output

Maze #1:
2 Cycles; the longest has length 16.

Maze #2:
There are no cycles.
