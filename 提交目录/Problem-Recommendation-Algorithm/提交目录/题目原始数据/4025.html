Description

　　话说石头剪刀布都有了必胜策略了。

　　后来，人们发明了新的剪刀石头布的方法

　　不过太复杂了，还是回到简单的剪刀石头布吧。
　　话说皮卡丘和草尼丘玩剪刀石头布，由于一只手玩有必胜策略，因此他们决定用两只手玩，现在规定，如果草尼丘的左手赢了皮卡丘的左手而且右手也赢了皮卡丘的右手那么草尼丘得两分，如果草尼丘的左手赢了皮卡丘的左手，右手平了皮卡丘的右手或者左手平了皮卡丘的左手，右手赢了皮卡丘的右手，则草尼丘得一分，否则草尼丘不得分。
　　但是由于草尼丘能够预判皮卡丘的出法，因此为了公平，他们都预先把自己每轮要出的写在n张纸上，然后每局他们就各拿自己写的一张纸出来进行比较，之后这张纸就作废了。
　　在写完之后，草尼丘不小心知道了皮卡丘每一张纸上面写的内容以及出的顺序，问草尼丘怎么出才能够使自己能得到的分数最大。

Input

第一行输入一个数T表示测试数据个数，对于每组测试数据，第一行输入一个数n，表示他们要进行n局比赛，之后输入两行，每行n个字符串，每个字符串长度均为2，每个字符串表示一种出法，第一个字母表示左手出的，第二个字母表示右手出的。而且字符串中只包含“R”，“S”，“P”三个字母，“R”表示出的石头，“S”表示出的剪刀，“P”表示出的布，例如“RS”表示左手出了石头，右手出了剪刀。对于这两行数据，第一行表示草尼丘的n张纸条，而第二行表示皮卡丘的n张纸条，并且皮卡丘会按顺序从第一张出到最后一张。

数据范围：
1<=n<=50

Output

对于每组测试数据，输出一个数，表示草尼丘能获得的最大分数。

Sample Input

3
1
RR
SS
2
SR PP
PR RS
4
PP PP PP PP
RR PR SS SP

Sample Output

2
1
3

HINT

对于第二组数据，草尼丘第一轮可以用PP对付皮卡丘的PR，得一分，第二轮草尼丘用SR对付皮卡丘的RS，得0分，所以总共得了一分。
