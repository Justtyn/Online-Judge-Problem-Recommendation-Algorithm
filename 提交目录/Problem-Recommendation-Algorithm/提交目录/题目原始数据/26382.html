Description

题目:
大学里的屌丝们都知道师大的妹子多，而Mummy学长刚好就在师大就读。
一天，Mummy在去惟义楼上课的路上看到这么一则通知:
我校是师范类院校，目前男女生比例约为3:7，惟义楼内设计男女卫生间使用情况比例为1:1，
经过调研和观察发现女生如厕明显不足，男生略有结余。为缓解这一矛盾，经过多方征
求意见后将男女卫生间拟做调整。
看明白了吧，男厕要改女厕了！！
Mummy有个特点，看到新鲜的东西总是想分享，这不，Mummy担心大家比赛太无聊了，于是就
有了这么一道题~~
Mummy了解到，在男厕改女厕之前，师大妹子中流传着"黄金屋"这么一个概念。
那什么是"黄金屋"呢？ 且听我细细道来。
惟义楼有好多女厕，但远远满足不了师大妹子，这就免不了要排队。妹子X要去如厕了，现在
有n个女厕供她选择，但是每个卫生间离她的距离D和排队的时间T都不一样。妹子X只去"黄金屋"
——就是女厕了，不过这个女厕要满足以下条件。
1.其他任意一个距离比该女厕近的女厕，排队时间都要比该女厕长。
2.其他任意一个排队时间比该女厕短的女厕，距离都比该女厕远。
例如:
妹子附近一共有有三个女厕A、B、C，这三个女厕离妹子的距离与排队时间依次为
A  300 100
B  100  300
C  400 200
其中A、B就是“黄金屋”
A: B比A近，B的排队时间更长
B: A的排队时间比B长，A的距离更远 ; C的排队时间比B长，C的距离更远
C 就不符合了

现在问题来了，妹子想要你写一个程序计算她附近所有"黄金屋"的个数。
妹子发话我们必须要执行对吧→_→




Sample Output

2

Input

有多组测试数据。每组测试数据的第一行是整数N (1 <= N <= 10000)，代表妹子附近女厕的个数。
接下来是N行(每一行表示一个女厕的信息),每行有两个整数D和T (1 <= D, T <= 10^11)，D代表该
女厕离妹子的距离，T代表该女厕所需的排队时间。当输入为0时表示输入结束，这组输入不用输出

Output

每组数据输出一个整数R，代表妹子附近"黄金屋"的个数。


Sample Input

5
300 100
100 300
400 200
200 400
100 500
0

Sample Output

2
