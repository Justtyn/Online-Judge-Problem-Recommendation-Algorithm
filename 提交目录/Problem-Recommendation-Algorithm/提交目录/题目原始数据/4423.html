Description

子惠是一个很狂热的萝莉控。一天，子惠听说有两只小萝莉八九寺真霄 和鹭之宫伊澄 出现在了迷失森林中，而且著名的把妹之手——小Ze也到了迷失森林外围，意图将两只小萝莉纳入后宫。于是子惠急匆匆的赶往迷失森林提前捕获。。啊不，拯救小萝莉。 子惠要赶在小Ze找到小萝莉之前将其营救。

可以营救到小萝莉当然很幸福啦，营救到真霄和伊澄分别可以提升子惠的幸福指数X和Y。

即使是小萝莉也是同性相斥的，两只小萝莉同时分布在迷失森林的东西两侧。子惠和小Ze同时只能去营救（捕获）其中的一只（？）。当子惠小Ze去了其中的一个方向后，不管有没有营救（捕获）到小萝莉，都会继续去另一个方向。当子惠小Ze不在同一个方向时，他们都能成功的营救（捕获）小萝莉。当子惠小Ze选择了同一个方向时，狭路相逢勇者胜，子惠有大小为P的概率成功营救小萝莉。也就是说假如子惠去了东侧，小Ze去了西侧，他们都能遇到小萝莉，然后子惠去西侧，小Ze去东侧，各自无所获。假如子惠小Ze都去了东侧，两人抢夺小萝莉，然后同时去西侧，继续抢夺小萝莉。

而子惠事先做好了研究，小Ze首先选择八九寺真霄的概率大小为Q。子惠希望能获得更高的幸福指数，请帮子惠计算，首先选择真霄能获得的幸福指数的期望高，还是选择伊澄能获得的幸福指数期望更高。

Input

第一行会输入一个数字T(1<=T<=1000)，代表输入数据有多少组。

         接下来有T组数据，每组数据占一行。

         每组数据有四个数字 X Y P Q。X,Y为整数， 1≤X, Y≤1000000000。P、Q为最多两位的小数，0≤P, Q≤1。

Output

每组数据输出一行，首先输出子惠应该做出的选择。如果选择真霄，请输出“zhenxiao”,如果选择伊澄请输出“yicheng”。空一格后输出子惠做出最好选择能获得的幸福指数的期望，输出保留四位小数。因为真霄比伊澄要小三岁，所以当两个选择的期望相同的时候，子惠会选择更小的真霄哦

Sample Input

3
2 1 0.5 0.5
2 1 0 1
7 7 0.32 0.16

Sample Output

zhenxiao 1.7500
yicheng 1.0000
zhenxiao 6.5968
