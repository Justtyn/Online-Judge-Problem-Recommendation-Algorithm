Description

    要到考试周了，老师们又开始忙起来了。助教也是一样，考试后，助教要批改试卷、统计分数、回答任课老师的各种问题……

DXY就是众多助教中的一个，某日，当他千辛万苦将成绩计算出来了，老师突然出现了在他的面前～老师想了解这次考试大家的情况，问了DXY一系列的问题，每个问题都差不多，都是想了解在某个区间段的分数的同学的最高和最低排名都是多少～

可怜的DXY已经忙的精疲力尽了～他希望你可以帮他回答这些问题、

Input

    输入数据有多组，第一行是一个整数N（0<N<=200000），表示总共有N个人的成绩，接下来N行给出了每个人的成绩Ai（0<=Ai<=100）。接下来有个整数M（0<M<=10）表示老师询问的次数。接下来M行，每行有两个整数X和Y（0<=X<=Y<=100），表示老师询问的区间范围[X,Y]。

N=0时输入结束。总数据不超过12组，排名没有并列。

Output

        对于每一个询问，输出两个整数，表示成绩在这个区间范围内的同学的最高排名和最低排名。如果没有同学的成绩在这个区间内，则输出两个0。
