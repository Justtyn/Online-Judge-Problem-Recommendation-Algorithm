Description

John有N部恐怖片子，编号0至N-1。第i部恐怖片的播放总长度是Length[i]分钟。现在John很累了，所以他可能在看某部电影过程中睡着了。唯一能让他一直保持不睡着的条件是：他受到的惊吓程度Level始终不低于某个值。一开始，John没看任何恐怖片之前的Level是定值74。他的Level随着时间的流逝而递减，每过一分钟就减少1。例如过了6秒后，Level值就减少0.1。一旦Level值小于-0.5，那么John就睡着了。每个恐怖片只有一个恐怖镜头，一旦John看到了这个恐怖镜头，他的Level值马上增加47。第i个恐怖片的恐怖镜头在该片子的第scary[i]分钟出现。
　 现在John想看完尽量多的恐怖片，他让你帮他安排恐怖片的编号排列，就是说，你要决定一个0至N-1的排列，按照这个顺序，John就可以在睡着前看完尽量多的恐怖片。如果有多种方案可以看完相同数量的恐怖片，输出字典序最小的序列。注意：一个恐怖片只有被John从头看到尾，才算看完，一旦John在看这个恐怖片过程中睡着了，那么他就没办法看完这部恐怖片，剩下还没看的那些恐怖片也无法再看到了。注意：电影是连续播放的，即一个电影结束后，瞬间就播放下一个电影。

Input

第1行：一个整数，N，表示恐怖片的数量(1≤N≤20)；
第2行：N个整数，第i个整数表示Length [i](2≤Length[i]≤474，对于0≤i≤N-1)；
第3行：N个整数，第i个整数表示scary[i](1≤scary [i]

Output

　 能尽量多地看完恐怖片的情况下，字典序最小的恐怖片的编号序列。

Sample Input

2
100 50
1 1
4
100 100 100 100
77 76 75 74

Sample Output

0 1
3 0 1 2
