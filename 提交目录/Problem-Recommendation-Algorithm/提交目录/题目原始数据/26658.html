26658: 我来打造合适的爪牙

Description

萨格拉斯，曾经的万神殿战士，伟大的泰坦。为了维持世界的秩序，不断地与诸多的恶魔战争。然而他终于还是开始怀疑自己的行为。在陷入了狂乱后，他堕落了。
他开始建立燃烧军团。
艾瑞达人，生活在阿古斯星球的民族，是萨格拉斯第一个选择引诱的种族。
艾瑞达人的三大领袖分别是基尔加丹，阿克蒙德和维纶。
萨格拉斯联络了他们，承诺给予他们无穷的力量与智慧。
几乎所有的艾瑞达人都选择向萨格拉斯效忠。
然而维纶预知了艾瑞达人的未来，他没有堕落成为萨格拉斯的爪牙，带领了其他不愿意臣服的艾瑞达人逃离了阿古斯。
他们来到了德拉诺，兽人的家园，开始了新的生活，并开始自称为德莱尼（艾瑞达语中的被放逐者）。


渴望加入燃烧军团的艾瑞达人是数不清的，然而他们仍然需要经过面试的考验，只有足够强大的艾瑞达人，才能成为燃烧军团的一员，才有资格为伟大的萨格拉斯效忠。

然而通往面试的神殿的路是极为狭窄的，一不小心就会跌入扭曲虚空，那些渴望成为萨格拉斯爪牙的艾瑞达人只能一个排在另一个的后面，而不能并排等待面试。

萨格拉斯原先关押那些恶魔的典狱官，拥有一种虚空转移的能力，可以交换两个相邻的艾瑞达人而不让他们跌入扭曲虚空。
但他毕竟不是萨格拉斯，如果两个相邻的艾瑞达人体重之和超过某个常数K，他便无力交换他们的位置。

现在，你，作为典狱官的爪牙，已经知道了当前所有等待面试的艾瑞达人的体重，以及常数K，你必须算出通过典狱官的虚空转移总共可能有多少种不同的排列（典狱官可以施放任意次虚空转移，他的法力值是无穷无尽的）。

Input

多组输入数据
每组数据包含三行。
第一行为一个整数n(1<=n<=100)，表示有n个艾瑞达人等待面试。
第二行为n个整数a1,a2,...,an（任意ai满足1<=ai<=1000000000）
第三行为一个常数k（1<=k<=1000000000），表示典狱官每次施放虚空转移的能力限制。

Output

对于每组输入数据，输出可能的排列数，由于答案可能非常大，你只需要输出答案对1000000007的模。

Sample Input

3
1 2 3
3

3
1 2 3
456

6
2 4 6 1 3 5
8

Sample Output

2
6
60
