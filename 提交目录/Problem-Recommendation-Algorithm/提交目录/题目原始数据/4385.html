Description

天游是StarCraft2 狂热分子，业余时间沉浸在星际争霸宏大的故事背景中。故事中的种族protoss利用幽能维持战争，他们通过建立幽能网络达到高效控制整个战区的能力。幽能网络是一个极为高效的网络，任意两个战区之间只需要维持一条幽能通道即可达到目的，在战争中protoss的暗黑执政官需要每一个战区必须能通过幽能网络达到任意的另一个战区。在战争中随着采集能量的累积，执政官能够利用到新的能源在战区间建立新的信道，或是改进一条已有信道的传输耗能。在战争中，能源是异常宝贵的，因此所有信道工作的总耗能最小是执政官们最关心的，并且每当改进或创建一条信道，执政官们都必须马上知道所有战区的联通情况，以及幽能网络的最小总耗能。你的任务是在执政官不断给出新的或改进的信道时，立即计算出战区的联通情况，和网络总耗能。

Input

测试数据的第一行将给出整数T，代表即将给出的T组数据。 对于每一组测试用例，首先给出一行两个整数n,m，分别代表n个战区，执政官的m次建立或改进。（ 0 < n < 1000 , 0 < m < 100000 ） 紧接着的m行输入，每行给出三个非负整数 x,y,z。表示在(x,y)间建立一条能耗为z的幽能通道。默认的，一条输入为对旧通道的覆盖当且仅当(x,y)之间存在已有的信道，且信道为双向传输( 1 <= x|y <= n, 0 <= z <= 200 )。

Output

对于每一组测试用例： 首先用一行输出 Case k:，k为当前第几组测试。 在接下来的m行中，作为对一次建立的响应，若当前n个战区可以互相可达，则给出网络总耗能，否则输出” The net still undone”。

Sample Output

Case 1: <br/>The net still undone <br/>The net still undone <br/>The net still undone <br/>13 <br/>11 <br/>8<br/>
