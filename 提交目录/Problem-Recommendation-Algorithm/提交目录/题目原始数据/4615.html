Description

今天是特殊的日子啊——新生赛，Skyboy在去的途中，就在想掏手机看时间时， 手机突然摔在地上。啪~ Skyboy心急想看看手机坏成怎样， 马上捡起手机。 眼看屏幕依然坚挺，并且在检查机身完，却没发现任何刮痕。 Skyboy立刻满怀大笑， 并自言自语的夸起自己的手机， 多耐摔，多牛B！不幸的是，就在他夸累后，开机发现手机触控不灵敏。

每触控一点，而与该点水平的或者垂直的任意点将不能被识别触控。只有锁屏（电源键）在开锁后才能重新恢复触控。Skyboy没想太多，只想知道，如果他顺序触控了多处， 哪些点才是有效的触控。但是，Skyboy编程学得不咋的，于是，他只好赶紧走到新生赛现场，请求你们帮忙解决了，不过Skyboy也不会为难你们的，他最多只会触控1000次就会锁屏的了，并且最多锁屏50次，手机会自动关机。（注：Skyboy的手机屏幕分辨率非常高, 是 1e8 * 1e8 的）

Input

测试数组有多组。每组数组有若干，每行两个整数x, y( 0<=x,y<=100,000,000).如果x ,y 分别为”Power” “button” 则表示锁屏并开锁， 如果x, y 分别为”Power” “off” 则程序结束。
