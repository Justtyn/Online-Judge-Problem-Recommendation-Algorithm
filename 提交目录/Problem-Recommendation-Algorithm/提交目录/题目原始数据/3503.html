Description

　　输入三角形三边边长，求三角形面积。

Input

　　一行：3个整数，空格隔开

Output

　　一行：一个数，表示三角形面积（小数点后保留两位）。

Sample Input

3 4 5

Sample Output

6.00

HINT



海伦公式求三角形面积：s=sqrt(d*(d-a)*(d-b)*(d-c))，其中d=(a+b+c)/2。



小数点后保留位数设置：printf("%.2\n",s);    保留两位
