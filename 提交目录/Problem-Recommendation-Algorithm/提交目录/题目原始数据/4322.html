Description

学校正在统计每门功课中同学的最高成绩，现在有n门功课，m个学生，你要求出这m个学生中每门功课的最高的成绩。
例如一共有2门功课，语文和数学。4个学生语文成绩分别为80，45，60，92，数学成绩分别为75，0，99，63。则求出语文成绩的最大值是92，数学成绩最大值是99。

Input

首先输入一个正整数n(n >= 1),表示一共有多少门功课。
然后输入m表示一共m个同学。
接下来n行输入每门功课m个同学的成绩。（成绩保证是非负整数）

Output

每输完一门功课的所有成绩，对应输出最高成绩。（每输出一个成绩都要换行）

Sample Input

3 5<br/>100 99 93 23 24<br/>42 35 54 43 34<br/>23 43 43 66 77<br/><br/>

Sample Output

100<br/>54<br/>77<br/>
