Description

         计算机是如何绘制图形界面的呢？

         我们知道屏幕是由一个个点构成的，在计算机显存中有一片区域代表着显示器，我们对那片显存的一个单元写入其对应的颜色值，屏幕上对应的点就会变成相应的颜色。这就是画点函数的由来。既然有了画点函数，那么我们要拓展画线函数，画圆函数，都是可以做到的，因为我们知道圆和线是可以拆成点的。利用这些，我们就能够画许多图形。

         假如我们正在制作一个动画演示程序，因为动画非常的复杂，如果每次都对整个屏幕更新会浪费掉大量CPU周期（即CPU计算时间）。因此我们决定采用脏矩形算法。例如这次在屏幕上(5,5)，(5,6)和(7,8)，我们认为我们弄脏了由(5,5)和(7,8)这两点作为对角线的矩形，这样我们就只对这个脏矩形内的部分进行绘制。(脏矩形的边总是平行于屏幕的某条边，即不会出现倾斜的矩形)

         （实际上这只是最简单的脏矩形算法，假设我们只对屏幕左上角和右下角进行了绘制，采用这种简单的脏矩形算法依然开销很大，不过简单的脏矩形算法也已经体现出了解决这种问题的核心思想。）

         现在我们告诉你，下一次在屏幕上绘图的所有点坐标。请你确定脏矩形的大小。

         （如果只有一个点，脏矩形的长宽都为1）


Input

第一行一个数字，代表有k个点坐标。k (1<=k<=100) （可能会出现重复的点）。

接下来k行，每行一个点坐标。（每个点坐标由2个数字x和y组成,分别代表水平方向坐标和竖直方向坐标）。

(0<=x<1024,  0<=y<768)



Output

两个数字a和b，分别代表脏矩形的宽度和高度。(宽度和高度以像素计算)(两个数字直接要有一个空格)


Sample Input

3
5 5
5 6
7 8

Sample Output

3 4
