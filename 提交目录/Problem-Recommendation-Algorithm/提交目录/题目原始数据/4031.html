Description

关于忍者，在翻墙或者爬山来执行任务时，用的较多的武器是苦无。话说苦无有几种作用：
一、是种爬墙的工具，因为有时候忍者需要贴在墙上进行任务，绳钩就显得有些不太方便了，用几把苦无配合着就可以插入墙壁用脚踩着用手拉着，末端的环里可以栓绳方便收回。　
二、是日本忍者所使用的一种暗器。
大概长这个样子的：
话说有个忍者要翻过一个悬崖，这个悬崖高度有n米，有两条可以爬上去的路，这两条路是平行的且垂直于地面的。而这两条路有一些地方是忍者无法到达的，因为这些地方的墙壁可能太坚固，导致忍者无法将苦无插入墙壁。
因此忍者每分钟可以有三种走法：
1.在同一条路上往前进1米
2.在同一条路上往后退1米
3.可以纵身一跃，跳到另一条路上，但是由于忍者轻功很好，每次都跳了m米高，也就是说，忍者的第三种走法是从一条路的第t个位置跳到另一条路的第t+m个位置。
当然忍者不能够处于那些有坚固墙壁的位置。
而且最近下大雨，那里的排水系统很差，所以水面以每分钟1米的速度往上暴涨，由于该忍者不会游泳，所以不能让他浸在水里，不然就被淹死了。
现在忍者位于第一条路的起始位置，而水面刚好在忍者的脚下，现在问忍者能否顺利翻过悬崖，注意，到达的位置超过悬崖顶端也算到达。

Input

第一行有一个数T表示测试数据的个数，对于每组测试数据，第一行输入n，m，分别表示悬崖的高度和跳跃的高度，之后输入两行，分别表示第一条路和第二条路的状况，其中‘-’表示该地方忍者可以到达，‘X’表示该地方忍者不可以到达，输入不保证第一条路的第一个位置一定是‘-’。
数据范围：
0<n,m<=10^5

Output

对于每组数据，如果忍者可以翻过悬崖，则输出“YES”，否则输出“NO”

Sample Input

2
7 3
---X--X
-X--XX-
6 2
--X-X-
X--XX-

Sample Output

YES
NO

HINT

第一组数据忍者原来在第一条路的第一个位置。

第一分钟跳到第二条路的第四个位置（移动方法3），水面在位置1

第二分钟往回移动一个位置，到达第二条路的第三个位置（移动方法2），水面在位置2

第三分钟再跳到第一条路的第六个位置（移动方法3），水面在位置3

第四分钟用第三种移动方法到达第二条路的第九个位置，直接翻过悬崖，输出YES

第二组数据忍者无法跳过悬崖，因此输出NO。
