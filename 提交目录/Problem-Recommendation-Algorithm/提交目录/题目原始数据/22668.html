题目正文：

         ACM的世界，那是由0和1组成的世界。自世界成立以来很多大神都喜欢二次元世界，因此ACM的世界是由0和1组成的二维世界（相当于由0和1组成的矩阵），而且为了完美，世界的大小是n*n的方形（宽度和高度不相等的话，那是相当不和谐！）

         随着ACM世界的发展，聪明的ACMer意识到偶数具有很多优秀的性质。因此，他们希望将ACM的世界改造成满足每个位置上下左右四个位置之和为偶数的世界。

         ACM的世界，允许让ACMer将任意一个位置的0改为1，但是不允许将1改为0。

         既然决定改造世界，ACMer自然是不会盲目动工的，他们要先利用计算机的力量来确定如何改造世界。

         现在，这项重担落在了你的头上，给你ACM世界现在的样子，如果能够成功改造，请输出最少要改动多少个位置。如果不能改造，输出-1。


Input

第一行一个数字n。代表ACM世界是n*n的。（1<=n<=15）

接下来n行，每行n个数（0或1），每个数代表ACM世界一个位置。


Output

一个数字。如果能成功改造，该数字是最少改动的位置。如果不能成功改造，该数字是-1。


Sample Input

3
0 0 0
1 0 0
0 0 0

Sample Output

3
